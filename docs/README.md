#자동차 경주 게임
## 기능 요구사항
- [X] 게임 플레이어에게 경주할 자동차의 이름을 입력 받는다.
    - [X] 이름은 쉼표(,)를 기준으로 구분한다.
    - [X] 이름은 5자 이하만 가능하다.
- [X] 게임 플레이어에게 시도할 회수를 입력받는다.
    - [X] 회수는 양의 정수로 입력받는다.
- [X] 사용자가 잘못된 값을 입력할 경우 예외를 발생시킨다.
  - [X] illegalArgumentException, illegalStateException 등과 같은 명확한 유형을 처리
  - [X] "ERROR" 로 시작하는 에러 메시지를 출력 후 그 부분부터 입력을 다시 받는다.
- [X] 자동차 객체 생성
    - [X] 자동차 객체는 0부터 9사이의 랜덤한 값에 따라 전진 여부를 판단할 수 있다.
      - [X] 랜덤한 값은 4이상 9이하면 전진, 0이상 3이하면 정지이다.
- [X] 자동차 경주를 진행한다.
    - [X] 자동차 경주 게임 결과를 출력한다.
    - [X] 게임 완료 후 누가 우승했는지를 알려준다.
      - [X] 우승자는 한명 이상일 수 있다.
- [X] 0부터 9까지의 랜덤한 숫자를 만든다.
