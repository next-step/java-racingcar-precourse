# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
- [X] 사용자 입력 예외처리
  - [X] 이름 5자 이하 검증(1자리 이상)
  - [X] 자동차 이동 횟수 입력 시 숫자 입력 검증
- [X] 자동차
    - [X] 양수라는 일급 객체를 가진 위치 값 추가
    - [X] 전진 기능 (멈춤 : 0 ~ 3, 전진 : 4 ~ 9)
- [X] 자동차 관리
    - [X] 자동차 이름은 쉼표(,)를 기준으로 입력받은 문자열을 나누어 생성
    - [X] 가장 멀리 간 자동차 조회
- [X] 사용자 입력
    - [X] 잘못된 입력 시 `IllegalArgumentException`, `IllegalStateException` 등 발생 시킨 후 재 입력
    - [X] 잘못된 입력 시 `[ERROR]`로 시작하는 에러 메시지를
      출력
    - [X] 게임 종료 조건 미충족 시 계속해서 사용자 입력 받기
- [X] 결과 출력
