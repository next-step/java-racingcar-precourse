# 프리코스 2주차

## 기능

- `MoveState` 움직임 상태 객체
  - Enum으로 상태를 표현하고, 각 Enum 상수가 의미하는 거리 객체를 맴버로 갖고 있음
  - 랜덤 인자값이 어떤 상태인지 확인해주는 정적 메소드를 갖고 있음 => `isIdle(int factor)`
- `Distance` 거리값에 명시적인 의미를 보여주기 위한 객체
  - `불변 객체`로 설계
  - 객체 생성이 `랜덤 인자에 의한 생성`과 `리터럴 값을 갖는 생성`으로 나뉨
  - 계산을 지원하는 메소드 add를 갖고 있음
    - `value 성격의 객체`는 다른곳에서 재사용시, 사이드 이펙트가 생길수 있기에 `새로운 객체를 반환` 
- `Name` 이름 객체
  - ~~5자 이하 유효성 검사~~ => 컨트롤러에 위임
- `Car` 자동차 객체
  - drive 메소드에서 랜덤 인자 생성 => 각 랜덤 인자는 자동차 객체마다 고유하게 작용되기에 각자 생성하는게 맞을것 같음 
- `Cars` 자동차들을 모은 객체 생성
  - 상기 자동차 객체를 담고 있는 일급 컬렉션
- RacingController
  - 입력받는 값 유효성 검사를 컨트롤러에서 처리
- `Error` 오류 메시지를 정의한 객체
- RacingService
  - 도메인 로직을 호출하기 위함
