# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록
* 자동차 이름 검증
* 자동차 대수 검증
* 자동차 이름 목록 검증
* 사용자 입력 자동차 이름들 검증
* 자동차 이름 얻기
* 
* 1 이상인지 검증
* 숫자인지 검증
* 레이싱 횟수 얻기
* 
* 자동차 전진 여부 적용
* 자동차 현재 주행거리 표시
* 모든 자동차의 현재 주행거리 표시
* 레이싱 1회 수행
* 
* 우승 자동차일 경우 이름 얻기
* 모든 우승 자동차들 이름 얻기
* 우승 자동차 탐색
*
* 자동차 우승 여부 얻기
* 우승 자동차 이름 얻기
* 게임 결과 생성
*
* 레이스 횟수 얻기
* 자동차 목록 생성
* 레이스 시작
* 게임 결과 출력
* 게임 시작
