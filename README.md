# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
- n대의 자동차 이름을 입력 받는다
  - 자동차의 이름의 길이는 5자 이하만 가능
  - n대의 자동차의 이름은 쉼표(,)를 기준으로 구분
- 몇번의 이동을 할 것인지 입력 받는다
- 사용자 입력 횟수 만큼 자동차들의 이동 여부 판단
  - 0~9 사이의 랜던값을 구한다
  - random 값이 4이상일 경우 전진, 3이하의 경우 제자리
- 이동 횟수 만큼 결과를 구한다
- 최종 우승자 결과를 구한다
