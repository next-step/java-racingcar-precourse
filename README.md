# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 기능 목록

#### Utility
- 문장 입력 받기 ✔
- 0~9 사이의 랜덤 숫자 반환 ✔
- 숫자 입력 받기
- , 기준으로 문자열 분리하여 반환

#### 자동차 게임 로직
- 자동차 이름은 5자 이하만 허용 ✔
  - 공백 입력 안됨
- 이동 횟수는 1 이상의 숫자만 허용 ✔
- 자동차 이름과 전진 횟수를 출력하는 기능 ✔
- 자동차의 이름과 전진 횟수 변수 래핑(모든 원시값과 문자열을 포장) ✔
  
- 자동차 전진 기능
    - 입력값이 0~3일 경우는 자동차 정지하는 기능 ✔
    - 입력값이 4~9일 경우는 자동차 전진하는 기능 ✔
    - 입력값이 0~9 사이의 값이어야 한다. ✔
- 자동차 경주 우승자 찾는 기능
