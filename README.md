# 자동차 경주 게임
## 규칙 방식   
* 가장 많이 전진한 차가 이김
* 0 ~ 9까지 Random값을 구한후, 4 이상 전진 

## 기능 구현 목록
* 차 생성 입력 기능
* 문자 5개 구분하는 기능
* 횟수 입력 기능 
* 랜덤 함수 생성 기능
* 차 이동 출력 함수 기능 
* 최종 우승자 판단 기능
* 최종 우승자 출력 기능

## 고민할 기능
* Tab, Space 고민할것
* Test, Function 함수 명확히 할 것
* 사용자 입력이 언제나 이상할 것을 파악할 것
* 에러나 로그를 좀 더 남기는 방법을 고민해볼지 생각 중
