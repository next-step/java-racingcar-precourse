# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
### 1. 자동차
* 각 자동차는 5자 이하의 이름을 가짐
* 각 자동차는 위치 정보를 가짐
* 0~9 사이의 랜덤 값이 4 이상이면 전진, 3 이하면 멈춤

### 2. 사용자 입력
* 자동차 이름 목록을 입력받음
* 쉼표로 구분하여 각 자동차를 생성
* 이름은 5자 이하만 가능
* 이동 횟수를 입력받음
* 잘못된 값 입력 시 [ERROR]로 시작하는 에러 메시지 출력 후 게임을 계속해서 진행할 수 있어야 함
    * 이름의 길이가 5자 이하가 아닌 경우
    * 이름이 null인 경우
    * 이동 횟수가 숫자가 아닌 경우

### 3. 우승자 판정
* 위치 정보가 가장 큰 값이 우승
* 우승자가 한 명 이상일 경우, 쉼표로 이름을 구분

### 5. 실행 결과 출력
* 이동 횟수만큼 반복하며, 각 회차별 결과 출력
* 최종 우승자 출력
