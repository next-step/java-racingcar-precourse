# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
* 자동차 이름을 입력 받는다.
* 입력받은 자동차 이름 유효성 검사를 진행한다.
  - 자동차 이름이 1~5자 사이의 문자로 구성되었는가?
  - 자동차 이름 중에 중복된 값이 있는가?
* 입력받은 시도 횟수의 유효성 검사를 진행한다.
  - 1 이상의 자연수인가?
* 모든 자동차에 대해 0~9 사이의 랜덤 숫자를 구한다. 
  - 숫자가 4이상이면 전진하고 4 미만이면 정지한다.
* 입력받은 시도 횟수 만큼 반복한다.
* 최종 Winner를 Return 한다.
