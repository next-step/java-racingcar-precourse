## 기능요구사항

* 주어진(m) 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다. 또한 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
  * 입력값 횟수를 입력 받는다.
    * 입력 유효값 : 턴수 m>0
    * 자동차 현재 상태값 : { 전진(go) : +1 } OR { 멈춤(stop) : 0 }
    * 자동차 횟수 진행 사항 저장
* 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
  * 자동차 이름 입력을 받는다. 이름은 5자 이하만 가능 하다
    * [o] 입력 유효값 : 5자 이하만 가능, 빈값 입력 불가
  * 입력된 자동차 이름은 출력 할때 사용 한다.
  * 자동차 이름은 쉼표(,)를 기준으로 구분 하여 여러대를 입력 할 수 있다.
* 전진하는 조건은 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우이다.
  * 무작위 값을 구한다.(9이하 양수)
  * 무작위 값은 4이상일 경우 -> 전진:횟수값+1
  * 무작위 값이 3이하일 경우 -> 멈춤:횟수값+0
* 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한 명 이상일 수 있다.
  * 우승자 출력
  * 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분한다.
* 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException를 발생시킨다
  * "[ERROR]"로 시작하는 에러 메시지를 출력 후 그 부분부터 입력을 다시 받는다.
* 아래의 프로그래밍 실행 결과 예시와 동일하게 입력과 출력이 이루어져야 한다.
