# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록

* 사용자로부터 입력받는 자동차 이름 예외처리
    * 같은 이름이 있는가?
    * 5자 이하인가?
    * 2대 이상인가?


* 사용자로부터 입력받는 이동 횟수 예외처리
  * 숫자인가?
  * 0보다 큰 숫자인가?
  

* 0에서 9 사이에서 random 값이 4이상일 경우 - 전진


* 0에서 9 사이에서 random 값이 3이하일 경우 - 정지


* 사용자가 입력한 값에 대한 자동차 경주 게임 결과를 구한다
    * 한 명 이상일 경우, 쉼표(,)로 표시
