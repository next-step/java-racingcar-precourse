# 자동차 경주 게임
## 개발 내용
* 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
* 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
* 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
* 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
* 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이 하의 값이면 멈춘다.
* 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한 명 이상일 수 있다.

## 구현할 기능 목록
* 정수 유틸(IntegerUtils) 객체
    * 0 에서 9 사이의 랜덤한 정수 생성 메소드

* CSV 유틸(CsvUtils) 객체
    * CSV 문자를 입력받아 문자 배열을 리턴하는 메소드
        * (예외) 입력된 문자가 비어 있는 경우

* 자동차 경주 설정(RacingCarConfig) 객체

* 경주 자동차(RacingCar) 객체
    * 입력받은 정수값에 의하여 전진 여부 결정하는 메소드
        * 입력된 정수 값을 체크해서, 4 이상일 경우 전진 3 이하 이면 멈춤 처리
        * (예외) 입력된 정수값이 0 에서 9 사이 값인지 확인
    * 다른 경주 자동차 객체와 위치값을 비교하는 메소드
    * 자동차명 인자로 받아 경주 자동차를 만드는 생성자
        * (예외) 입력된 자동차명 중 비어 있는지 확인
        * (예외) 입력된 자동차명 중 입력 길이를 초과한지 확인

* 자동차 경주 관제(RacingControlTower) 객체
    * 참여할 경주 자동차들을 CSV 문자로 인자로 받아, 자동차 경주 정보를 생성하는 생성자
        * (예외) 입력된 자동차명 중 중복된 값이 있는지 확인
    * 자동차 경주에 대한 결과, 승자 정보를 리턴하는 메소드

* 자동차 경주 게임(RacingCarGame) 객체
    * 게임 시작 전 준비 메소드
    * 게임 시작 메소드
    * 게임 결과 출력 메소드

* 게임 실행(GameApplication) 객체
    * 실행 메소드
