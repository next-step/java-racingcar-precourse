# 자동차 경주 게임
## 구현 기능 목록

---
- 경주할 자동차의 이름을 입력
    - 자동차 대수 n은 0보다 커야 한다.
    - 자동차의 이름은 1-5자이하 여야 한다.
    - 자동차의 이름은 중복될 수 없다.
- 시도할 횟수 입력
    - 정수의 범위 내에서 입력 한다.
    - 0 보다 커야 한다.
- 경주 진행
    - 자동차의 랜덤 숫자 생성
    - 랜덤 숫자에 따른 전진/멈춤
      - 4이상은 전진, 3이하는 멈춤
    - 자동차의 위치 변경
    - 자동차들 위치 표시
- 최종 우승자 표시
    - 가장 앞에 있는 자동차 이름 출력
    - 공동 우승의 경우는 쉼표(,)로 이름을 구분하여 출력


## 2주차 주안점

---
- 도메인 로직 분석
- 객체 추출
- 패키지/클래스/메소드 분리  
  - controller - 레이싱 게임 요청 처리 패키지
  - domain - 도메인 로직 패키지
  - view - UI 패키지
- 일급 컬렉션의 활용
- 모든 원시값과 문자열을 포장(Wrapping)
- TDD를 시도한다.
- 프로그램 제약사항을 준수한다
