# java-racingcar-precourse

## 구현할 기능
- [ ] 경주할 자동차 이름 입력받기
  - 자동차들은 쉼표로 구분
  - 이름은 5자 이하만 가능
    - [ ] 잘못된 값 입력 시 IllegalArgumentException 처리
- [ ] 시도할 횟수 입력받기
  - [ ] 잘못된 값 입력 시 IllegalArgumentException 처리
- [ ] 시도할 횟수만큼 자동차 이동
  - [ ] 각 자동차마다 0~9사이 랜덤 값을 구한 후, 무작위 값이 4이상일 경우 전진
- [ ] 우승한 자동차 구하기(여러 자동차 가능)
