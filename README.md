# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능요구
### 기능
1. 차량의 명칭 부여
   1. ,를 기준으로 자동차의 이름을 구분
      1. 이름이 없는경우 다시 입력
   2. 5자 이하의 이름만 부여 가능
      1. 5자 이상이 있는 경우 다시 입력.
2. 시도횟수 입력 기능
   1. 주어진 횟수동안 명칭이 부여된 n대의 자동차가 전진, 멈춤
      1. 입력된 값이 숫자인지 확인
   2. 랜덤기능을 활용해서 0~9까지의 숫자를 봅음.
      1. 랜덤 값이 3이하의 경우 멈춰있음.
      2. 랜덤 값이 4이상일 경우 전진함.
3. 가장 많은 전진을 한 차량이 우승.
4. 잘못된 값 입력시 ERROR로그 출력 후 입력을 다시받음.
### 출력
1. 자동차이름을 받기위한 안내문.
   1. 잘못된 값 입력시 ERROR 출력 및 재입력 안내.
2. 시도횟수 입력 안내문.
   1. 잘못된 값 입력시 ERROR 출력 및 재입력 안내.
3. 실행결과 출력
   1. 제목 출력
   2. 게임이 매판 진행후의 전진 또는 스탑에 대한 결과 출력
   3. 최종 우승 결과에 대한 출력
