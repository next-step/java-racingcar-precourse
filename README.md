# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현 목록

- 1 ~ 9 까지 랜덤한 수 생성 기능
- 한칸 전진하는 기능이 있는 자동차와 일급 컬렉션 구현
- 입력이 4 이상일 경우 이동하고 아닐 경우 이동하지 않는 기능 구현
- 차들의 이름을 , 로 구분하여 입력받고 그 수만큼 생성하는 기능 및 5자 초과인 경우 예외처리
- 횟수를 입력받아 모든 차들에게 횟수만큼 이동명령을 주는기능 구현
