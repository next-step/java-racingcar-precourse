# 자동차 경주 게임

## 기능 요구 사항

* 사용자 입력
    * 각 자동차에 이름 부여
        * 쉼표(,)를 이용하여 구분
        * 자동차의 이름은 5자 이하만 가능
    * 자동차가 이동가능한 Round의 수 입력
    * 사용자가 잘못된 값을 입력하는 경우
        * [ERROR]로 시작하는 에러메시지를 출력 후, 사용자가 다시 입력
* 자동차 이동
    * 0~9 사이의 random값 중,
        * 값이 4이상일 경우 전진
        * 값이 3 이하인 경우 멈춤
    * 전진하는 자동차 출력할 때, 자동차 이름을 같이 출력
* 게임 종료
    * 사용자가 입력한 Round의 수가 끝나면 제일 많이 이동한 우승자 출력
    * 우승자가 한명 이상일 경우, 쉼표(,)로 이름 구분하여 출력
