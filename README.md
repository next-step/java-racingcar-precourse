# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 요구사항
- [x] 자동차
  - [x] 자동차는 이름과 전진 횟수를 가지고 있다.
  - [x] 이름은 5자 이하만 가능하다.
  - [x] 5자 이상 이름으로 자동차 생성시 [ERROR] 메세지를 반환한다.
  - [x] 0~9 사이의 random값에서 random값이 4이상일 경우 전진한다.
  - [X] 0~9 사이의 random값에서 random값이 4미만일 경우 정지한다.
 
- [x] N대의 자동차
  - [x] 자동차의 이름은 쉼표(,)를 기준으로 구분한다.
  
- [x] 경주 게임
  - [x] 게임을 시도할 회수만큼 게임을 진행한다.
  
- [ ] 우승자
  - [ ] 자동차 경주게임 완료 시 우승자를 반환한다.
  - [ ] 우승자가 한 명 이상일 경우 쉼표(,)로 이름을 구분해 반환한다.

- [ ] UI
  - [ ] 경주할 자동차 이름을 입력한다.
  - [ ] 잘못된 이름 입력 시 [ERROR] 메세지 출력 후 이름을 재입력 한다.
  - [ ] 시도할 회수를 입력한다.
  - [ ] 자동차 이름과 전진 실행 결과를 출력한다.
  - [ ] 최종 우승자를 출력한다.
