# 자동차 경주 게임
## 도메인 규칙
- [ ] 자동차들이 있다.
  - [ ] 여러개의 자동차를 갖는다.
  - [ ] 전체 전진이 가능하다.
- [ ] 자동차가 있다
  - [ ] 자동차 이름이 있다.
  - [ ] 전진이 가능하다.
  - [ ] 전략을 통해 이동이 가능하다.
    - [ ] 기본 전략은 랜덤전략이다.
      - [ ] 0~9까지의 값 중 4이상이면 전진이 가능하다.
- [ ] 자동차 이름들이 있다.
  - [ ] 자동차 이름들을 생성할 떄 String으로 생성이 가능하며, 구분자로 나눌 수 있다. 
  - [X] 자동차 이름이 있다.
      - [X] 5글자를 넘길 수 없다.
- [ ] 우승자가 있다.
  - [ ] 우승자는 여러명일 수 있다.
  - [ ] 최고점수 기반으로 우승자를 구해낸다.
