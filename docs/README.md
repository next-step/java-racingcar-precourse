# 구현할 기능 목록

* 경주할 자동자 이름 입력받기
    - 입력받을 때 쉼표(,)로 구분
    - 이름은 5자 이하만 가능
    - 전진하는 자동자 출력할 때 자동자 이름을 같이 출력
* 자동자를 몇 번 이동할 것인지 입력받기
* 자동차 전진 조건: 0~9 사이에서 random 값에 따라 이동
    - random 값이 4 이상: 전진
    - random 값이 3 이하: 멈춤
* 경주 게임 완료 후 우승한 자동차 이름 출력
    - 공동 우승자는 쉼표(,)로 구분
    - 단독 우승자는 이름만 출력
* 예외 처리
    - 사용자가 잘못된 값을 입력할 경우
        + IllegalArgumentException 발생
        + "[ERROR]"로 시작하는 메시지 출력
        + 다시 입력 받기
