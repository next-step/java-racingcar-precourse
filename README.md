# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록
* 0-9 사이 랜덤값 생성
* 자동차는 이름, 위치 속성이 있고, 랜덤값 생성 가능
* 이름은 5자 이하만 가능  
* 자동차는 생성한 랜덤 값에 따라 전진 또는 유지
* 위치 시각화
* 자동차의 이름 및 위치 출력
* 자동차 이름 입력받아 자동차 생성
* 메시지 출력 및 시도 횟수 입력받아서 반복 처리
* 우승자 구하기
* 시도 횟수에 따른 우승 결과 출력
