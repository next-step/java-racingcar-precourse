# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 시나리오

- 사용자는 경주할 자동차 이름 리스트를 입력합니다.
  - 자동차 이름은 쉼표(,) 로 구분하며 최대 5 글자입니다.
- 사용자가 시도할 횟수를 입력합니다.
- 매 횟수마다 각 자동차는 0 ~ 9 의 랜덤값을 받아서 값이 4 이상이면 전진하고 3 이하면 멈춥니다.
- 각 자동차가 움직인 거리를 이름과 함께 출력합니다.
- 최종 우승한 자동차를 구합니다.

## 구현 기능

### 1. Car

- 자동차의 이름은 최대 5 글자
- 0 ~ 9 사이의 값을 받아서 4 이상이면 전진하고 3 이하면 멈춤

### 2. Input

- 자동차 리스트 입력: 쉼표로 구분
- 시도할 횟수 입력

### 3. Output

- 각 자동차가 움직인 거리 출력
- 최종 우승자 출력
