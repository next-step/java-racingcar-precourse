# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
* 횟수 입력 받기
* 횟수만큼 play 반복하기
* play마다 각 자동차 전진 또는 멈춤 기능
* 자동차 이름 입력받기(쉼표로 구분)
* 자동차 리스트 생성하기
* 이름 입력받은 대수 만큼 자동차 생성하기
* 자동차 생성 시, 이름 5자 구분하기
* 0 ~ 9 사이 랜덤값 구하기
* 4이상 전진 기능 구현하기
* 완료 후, 우승자 판정하기
* 우승자 여러명인 경우, 리스트에 담기
* 우승자 출력은 쉼표로 구분하기
* 잘못된 입력값 받았을 때 [ERROR] 메시지 출력하고 다시 입력받기
