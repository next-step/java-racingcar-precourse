# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
* 경주할 자동차 이름을 입력받는 기능
* 시도할 회수를 입력받는 기능
* 0에서 9사이 랜덤 값을 구하는 기능
* 랜덤 값이 4 이상이면 전진하고 3 이하의 값이면 멈추는 기능
* 실행 현황을 출력하는 기능
* 최종 우승자를 가리는 기능
* 실행 결과를 출력하는 기능
