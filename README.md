# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 요구사항

- 자동차
  - n 대 존재
  - 5자 이하의 이름이 존재
  - 4 이상의 값이 들어오면 전진
  - 3 이하의 값이 들어오면 멈춤
  - 자동차가 이동한 길이는 `-` 의 연속으로 표현

- 사용자 입력
  - 자동차 이름을 쉼표로 구분하여 입력
  - 몇 번의 이동을 할지 입력
    
- 경주 게임
  - 사용자가 입력한 이동 횟수 동안 자동차 경주 시작
  - 랜덤한 수를 구해서 자동차를 차례로 이동시킴
  - 경주 게임을 완료한 후에는 우승자를 표시 (공동 우승 가능)
