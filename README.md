# 자동차 경주 게임
## 진행 방법

---
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정

---
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록

---
### 1) 경주용 자동차 생성
* 경주용 자동차
1. 자동차 생성시 이름 부여
2. 자동차 이름은 5자 이하
3. 생성할 자동차 이름의 구분은 쉼표(,)로 구분
4. 자동차는 n대 생성 가능 (구분자이용)   
5. 예외처리
    1. 이름입력시 6자이상일 경우 
    2. 이름이 공백일 경우

### 2) 생성된 자동차의 이동
1. 자동차의 이동조건 : 4 이상일 경우
2. 자동차의 정지(미이동)조건 : 4 미만일 경우
3. 이동조건값은 랜덤 수 : 0~9 범위 내 랜덤 수 생성
4. 이동 Try 횟수 : 사용자 입력 n 번 

### 2) 생성된 자동차로 레이싱 게임
1. 사용자 입력 자동차 이름으로 경주용 자동차 n개 생성
2. 사용자 입력 Try횟수로 경주용 자동차 전체 이동 시도
3. 경주용 자동차별 Try횟수 단계별 이동카운트 저장
4. Max이동한 우승 자동차 찾아서 출력
5. 예외처리
    1. 시도횟수 0번일 때, 숫자 아닐 경우
    2. 자동차 이름 쉼표 아닌 구분자 및 공백, 한글등 유효하지 않은 값일 경우
