# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
* 사용자로부터 입력 받을 시도 횟수
    * 시도 횟수는 1회 이상인지 검증한다.
    

* 사용자로부터 입력 받을 자동차 이름
    * 자동차 이름은 쉼표(,)를 구분자로 하여 분리할 수 있는지 검증한다.
    * 자동차가 2대 이상인지 검증한다.
    

* 자동차 이름
    * 자동차 이름은 공백이 아닌 1자 이상 5자 이하여야 한다.


* 자동차의 전진 여부룰 결정하는 임의의 숫자
    * 임의의 숫자는 0 이상 9 이하만 가능하다.
    

* 자동차
    * 임의의 숫자가 4 이상일 경우 전진한다.
    * 임의의 숫자가 4 미만일 경우 정지한다. 


* 자동차 경주의 진행
    * 시도 횟수마다 자동차들의 이동 상황을 출력해야 한다.
    

* 자동차 경주의 최종 결과
    * 우승자, 즉 경주에 참여한 자동차들 중 가장 많이 이동한 자동차의 위치를 구한다.
    * 우승자와 동일한 거리만큼 이동한 공동 우승자를 구한다.
