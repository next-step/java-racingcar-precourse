# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 용어
 - 전진 시드값, SeedForMove : 자동차가 전진할지 말지 결정하는 0 ~ 9의 값. 4이상이면 전진 3이하면 멈춤. 

## 기능 목록
 - 전진 시드값 수신 시 입력받는 값이 올바르고, 전진시킬 값인지 판단하는 기능
 - 자동차 생성 및 정보를 출력 하는 기능
 - 자동차를 이동시키는 기능
 - 자동차의 이름이 5자 이하인지 validation 하는 기능
 - 자동차의 이름들을 입력받아 자동차를 생성하는 기능
 - 자동차의 이름들 사이에 쉼표가 존재하는 지, 중복이 없는지 validation 하는 기능
 - 자동차들의 이동횟수를 입력받아 전체 자동차를 반복 전진&멈춤 시키는 기능
 - 우승자를 출력하는 기능
