# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


---
## 구현할 기능 목록
- 입력 Console
    - 자동차 이름 입력
        - 5자 초과일시 예외 처리(숫자, 특수문자, 공백 허용)
    - 몇 번 이동할 것인지 입력
        - 0이하의 숫자, 문자 입력 시 예외 처리


- 시도 결과
    - 자동차별 랜덤 숫자 생성
    - 전진, 후진 비교 기능


- 출력 Console
    - 실행 결과(전진, 후진) 출력
    - 최종 우승자 출력
