# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 기능 목록
## UI
* 자동차의 이름을 입력 받는 기능
* 이동할 횟수를 입력 받는 기능
* 이동 단계마다 차 이름과 이동 상태를 출력 하는 기능
* 우승자를 출력하는 기능

## DOMAIN
* 전체 게임 진행을 주관하는 기능
* 자동차 이동 기록을 관리하는 기능
* 자동차를 이동할 횟수 만큼 이동 시키는 기능
* 랜덤 숫자 1개를 생성해 전진 여부를 결정하는 기능
* 자동차 게임이 완료 된 후 우승자를 찾는 기능
