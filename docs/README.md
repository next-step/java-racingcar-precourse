## 요구사항
1. 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
2. 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력해야한다.
3. 자동차 이름은 쉼표(,) 기준으로 구분하며 이름은 5자 이하만 가능하다.
4. 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
5. 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일경우 전진, 3이하면 멈춘다.
6. 자동차 경주 게임을 완료한 후 누가 우승했는지 알려준다.
7. 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
8. 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException 을 발생시키고 [ERROR] 로 시작하는 메시지를 출력 후 그 부분부터 입력을 다시받는다.
9. Exception 이 아닌 IllegalArgumentException, IllegalStateException 등과 같은 명확한 유형을 처리한다.
10. 일급콜렉션을 활용해 구현한다.
11. 모든 원시값과 문자열을 포장한다.

## 구현할 기능 목록
1. 자동차 생성
- 이름을 가지고 있다.
    1) 이름은 5자 이하만 가능하다.
- 위치값을 가지고 있다.
- 엔진을 가지고 있다.
    1) 엔진 충전은 0-9 사이의 숫자로 가능하다.
    2) 에너지가 4 이상일 경우에 움직일수 있는 상태가 된다.
- 전진 기능이 있다.
    1) 엔진 에너지가 충분하면 위치값을 증가시킨다.
- 위치값으로 서로간의 비교가 가능하다.

2. 사용자 입력받기
- 사용자는 자동차 이름을 입력할 수 있다.
    1) 자동차 이름은 쉼표(,) 기준으로 구분하며 5자 이하만 가능하다.
- 사용자는 몇번의 이동을 할 것인지 입력할 수 있다.
    1) 숫자가 아닌경우
    2) 양수가 아닌경우
3. 출력하기
    - 이동 횟수만큼 출력한다.
    - 전진하는 자동차 이름을 같이 출력한다
        1) 자동차 이름을 같이 출력한다.
        2) 이동 횟수만큼 "-" 가 늘어난다.
4. 경주 게임    
   - 입력받은 자동차 이름 숫자만큼 자동차를 생성
     - IllegalArgumentException 발생 후 "[ERROR]" 로 시작하는 메시지 출력 후 다시 입력 받는다.
   - 입력받은 이동 횟수만큼 자동차 move
   - 끝나면 종료한다.
       1) 누가 우승했는지 알려준다.
       2) 한 명 이상일 경우 쉼표(,)로 이름을 구분해 출력한다.
