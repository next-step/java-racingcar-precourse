# 자동차 경주 게임

## 기능 요구사항
1. 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
2. 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
3. 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
4. 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
5. 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다.
6. 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
7. 사용자가 잘못된 값을 입력할 경우 "[ERROR]"로 시작하는 에러 메시지를 출력 후 입력을 다시 받는다.

## 기능 목록
1. ✅자동차
    - ✅1~9의 랜덤값을 생성
    - ✅random 값이 4 이상일 경우, 전진이 가능 한가?
    - ✅random 값이 3 이하일 경우, 정지가 가능 한가?
    - ✅이름이 5자 이상일시, 오류가 발생하는가?
2. ✅레이싱게임
   - ✅잘못 입력한 경우 오류 발생
   - ✅쉼표(,)를 기준으로 자동차를 생성한다.
   - ✅주어진 횟수동안 Racing을 반복한다.
3. ✅메인
   - ✅자동차 이름을 입력받는다.
   - ✅시도할 횟수를 입력받는다.
   - ✅게임이 시작되면 Racing을 반복한다.
   - ✅Racing이 끝났다면 우승자들을 출력한다.
4. 리팩토링
   - View 부분 분리
   - 원시값 문자열 포장
   - 일급 콜렉션
   - ...
