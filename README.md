# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록
- [ ] 도메인 개발
- [ ] 자동차에 이름을 입력하는 기능 구현
  - [ ] ',' 로 구분하여 자동차별로 이름을 입력하는 기능 구현
  - [ ] 각각 이름의 길이가 5자 이하인지 확인
- [ ] 사용자가 입력횟수를 입력하는 기능구현
  - [ ] 입력횟수는 1이상의 숫자인지 확인하는 기능 만들기
  - [ ] 사용자가 숫자가 아닌값을 입력시 "[ERROR]"로 시작하는 에러 메시지 출력하는 기능 만들기
- [ ] 전진 횟수 입력시 전진하는 자동차 만들기
  - [ ] Random값이 4이상이면 전진하는 기능 만들기
  - [ ] Random값이 3이하이면 정지하는 기능 만들기
- [ ] 최종 자동차 전진값으로 우승후보 판단하는 기능 만들기
- [ ] 우승자 출력 기능 만들기
   - [ ] 우승자가 1명 이상일 경우 ','로 구분해 출력하는 기능 만들기
