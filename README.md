# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 자동차 이름 입력
  * 자동차 이름(들)은 한번에 입력하며 ','로 구분
  * 5자리 이상 입력 시 '[ERROR]' 문구 메시지 출력
* 게임 진행 횟수 입력
  * 숫자가 아닌 문자 입력 시 검증 처리
  * 입력된 숫자만큼 게임 실행 진행
* 자동차 생성
  * Car 객체는 이름, 이동결과, 랭킹을 보유
  * 랜덤값의 따라서 계산된 값을 이동결과의 누적하여 저장
    * 4이상일 경우 전진(-)
    * 3이하일 경우 멈춤()
가
  * 각 자동차 이동 결과의 대한 랭킹 계산
  * 계산된 랭킹의 대해 우승자 출력
  * 우승자가 1명 이상일 경우 (,) 로 구분 처리
