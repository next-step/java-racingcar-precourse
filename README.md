# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현 목록
* 사용자로부터 입력 받는 자동차 이름 예외 처리.
	- 자동차 이름이 5자 이하인가?
	- 쉼표(,)를 기준으로 자동차 구분이 가능한가?
	- 1개 이상의 자동차가 입력 되었는가?	
* 사용자로부터 입력 받는 실행 숫자 예외 처리.
	- 1보다 크거나 같은 숫자인가?	
* 1~9의 숫자 중 랜덤 숫자를 구한다.
* 랜덤 값이 4이상인 경우 - 전진
* 랜덤 값이 3이하인 경우 - 정지
* 사용자에게 입력받은 숫자와 전진한 숫자가 같은 경우 - 골
* 사용자가 입력한 값에 대한 실행 결과를 구한다.
