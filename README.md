# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
1. 자동차 기능
    * 이름을 가질 수 있다
      * 자동차의 이름은 5자 이하여야 한다
    * 아래의 조건에 따라 전진하는 기능을 가진다
      * 조건은 0 ~ 9 사이의 랜덤값을 구한 후 값이 4 이상인 경우 전진, 3 이하인 값이면 전진하지 않는다

2. 자동차그룹 기능
   * n대의 자동차를 가질 수 있다
   * 자동차들의 전진 기능을 실행시킨다
   * 우승자를 선정한다

3. 게임 뷰 기능
   * 자동차의 이름을 입력 받는다
     * 이름은 쉼표를 기준으로 구분하여 입력받는다
   * 몇 번의 이동을 시킬지 입력 받는다
   * 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException를 발생시키고, '[ERROR]'로 시작하는 에러 메세지를 출력 후 그 부분부터 입력을 다시 받는다
   * 자동차의 현재 위치를 출력한다
   * 우승자 정보를 출력한다
     * 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다 

4. 컨트롤러 기능
   * 아래의 순서로 게임이 진행되도록 한다
     * 경주할 자동차 이름을 입력 받는다
       * 에러가 발생하면 다시 입력 받는다
     * 시도할 횟수를 입력 받는다
       * 에러가 발생하면 다시 입력 받는다
     * 입력받은 횟수만큼 이동, 위치 출력 작업을 반복한다
     * 우승자 정보를 출력한다
