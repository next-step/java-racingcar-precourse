# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 구현 기능 목록
### 랜덤 숫자 생성
* 사용자가 입력한 숫자 범위와 추출할 갯수를 기반으로 랜덤 숫자 배열을 생성한다.

### 자동차 
* 자동차는 5글자 이하의 이름을 가진다. 
* 자동차는 4이상 숫자가 주어지면 전진 3 이하면 멈춘다.
* 모든 원시값과 문자열을 포장한다.

### 자동차 게임
* 자동차에게 전진/정지 결정을 위한 값을 넣는다.
* 자동차의 현재 전진 상태를 보여준다.
* 가장 높은 점수인 자동차(들)을 보여준다.

### UI
* 사용자가 입력한 자동차 이름을 자동차 리스트로 변경한다.
* 사용자가 입력한 시도 횟수만큼 수행한다.
* 실행 결과를 시도한 횟수만큼 제공한다.
* 최종 우승을 나타낸다.
