# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 구현할 기능 목록
  1. 자동차
     - 속성 : 이름
     - 전진 기능 : 이동 가능 여부는 자동차의 기능
  1. Car 를 만들 Factory
     - , 분리하고 5글자 확인 
     - 자동차를 생성한다
  1. 게임에 참여하는 자동차 클래스
     - 게임 중 자동차의 위치를 저장할 CarPosition 
     - 게임 중인 자동차를 표현할 PlayingCar
     - 자동차 간 누가 앞서고 있나? 확인할 메소드 구현
     - 자동차의 현재 정보를 출력하는 메소드 구현
  1. Game 정의 
     - 게임에 참여하는 자동차들의 목록
     - 이동 횟수 만큼 모든 자동차를 이동한다
     - 현재 기준 우승자 목록을 반환한다
  1. 사용자로부터 이름 목록을 입력 받는다.
     - 입력받는다     
  1. 게임 진행 구현
     - 이름을 입력 받아 자동차 리스트를 생성한다
     - 시도할 횟수를 입력 받는다
     - '게임'을 만든다.
     - 게임을 실행한다
     - 게임 결과를 출력한다
