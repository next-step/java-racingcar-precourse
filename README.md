# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 요구사항
- 0~9 사이의 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
- 전진하면 자동차의 이동거리를 추가한다.
- 이동거리가 가장 긴 자동차(들)를(을) 찾는다.
- 이동거리가 가장 긴 자동차의 이동거리를 찾는다.
- 자동차를 이동 시킨다.
- 우승차량인지 확인한다.
- 우승자 리스트를 생성한다.
- 우승한 자동차를 출력한다.
- 사용자로부터 시도 횟수 입력
  - 숫자인가?
- 사용자로부터 자동차 이름 입력(쉼표(,)를 기준으로 구분)
  - 각 이름은 5자 이하인가?
