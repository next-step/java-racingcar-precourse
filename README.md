# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 요구사항 정의
### 입력
- [ ] 경주할 자동차들의 이름을 부여 받는다
  - [ ] 각 자동차의 이름은 쉼표로 구분한다
  - [ ] 각 자동차의 이름은 5자 이하여야 한다
  - [ ] 경주는 2명 이상이 해아한다
- [ ] 이동할 수 있는 회수를 입력받는다
  - [ ] 해당 회수는 숫자여야 한다

### 출력
- [ ] 전진하는 자동차를 각 회수마다 출력한다
  - [ ] 자동차의 이름을 출력한다
  - [ ] 자동차의 전진상태를 출력한다
- [ ] 우승자의 이름을 출력한다
  - [ ] 우승자가 2명이상일 경우 쉼표로 구분하여 출력한다
- [ ] 사용자가 잘못된 값을 입력하면 "[ERROR]" 로 시작하는 메시지를 출력한다

### 기능
- [ ] 자동차를 정의한다
  - [X] 자동차는 이름을 입력받아 생성한다
  - [X] 자동차의 이름은 비어있으면 안된다
  - [X] 자동차의 이름은 5글자 이하여야 한다
  - [ ] 자동차는 현재의 위치 상태를 저장한다
- [ ] 자동차가 전진하거나 멈춘다
  - [ ] 입력받은 값이 4이상이면 전진한다
  - [ ] 입력받은 값이 3이하이면 멈춘다
  - [ ] 0 ~ 9 사이의 값만 입력받을 수 있다
- [ ] 사용자가 잘못된 값을 입력하면 예외를 발생시킨다
  - [ ] 예외를 전달하고 입력을 다시 받는다
