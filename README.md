# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 사용자는 경주할 자동차의 이름을 입력한다.
  * 이름 입력 시 쉼표로 구분한다.
  * 자동차는 두 대 이상이어야 한다.
* 사용자는 시도할 횟수를 입력한다.
* 전진한 자동차의 이름과 이동거리를 출력한다.
  
* 자동차는 전진할 수 있다.
  * 각 자동차는 주어진 횟수 마다 0~9 사이의 난수를 입력받는다.
  * 난수의 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
* 자동차는 이름을 가진다.
  * 5자 이하여야 한다.
* 자동차는 이동거리를 가진다.
  
* 자동차 경주 게임을 완료한 후 누가 우승했는지 알려준다. (1명 이상)
