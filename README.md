# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

---
## 기능 목록
* 자동차 이름 입력
    * 쉼표(,)를 기준으로 구분
    * 유효성 검사
      * 자동차 이름 길이 확인
      * 자동차 이름 중복 확인
* 이동 횟수 입력
    * 숫자인지 확인
* 0에서 9까지의 숫자 랜덤 값
    * 자동차 이름 입력 개수 만큼 반복
* 4이상의 값 전진, 3이하의 값 멈춤
* 우승자 확인
