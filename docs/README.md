
# 기능

## 개요
더 먼 거리를 이동한 자동차가 이기는 게임.

* 입력받은 턴의 횟수 만큼 전진을 시도할 수 있다. 

## 상세
[시작 - 자동차 이름 입력(사용자) - 턴의 횟수 입력(사용자) - 모든 턴을 플레이 - 승자 결정 - 종료] 의 순서를 가진다.

* 사용자로부터 2개 이상의 자동차의 이름을 입력받는다.
  * 쉼표로 구분된다.
  * 자동차 이름은 5자 이하여야 한다.
* 사용자로부터 턴의 횟수를 입력받는다.
  * 횟수는 0 이상이어야 한다.
* 모든 자동차는 입력받은 횟수만큼 전진을 시도할 수 있다.
  * 6:4의 무작위 비율로 전진할지 멈춰있을지 정한다.
* 모든 턴이 끝나면, 각 턴마다 모든 자동차가 이동한 거리를 -로 출력한다.
  * 출력 예시
    * A: ---
    * B: --
* 마지막으로 최종 우승한 자동차를 출력한다. 마지막 턴까지 가장 멀리 이동한 자동차가 우승한 자동차이다.
  * 출력 예시
    * 최종 우승자: A, B
* 최종 우승자를 출력하면 게임을 종료한다.
* 잘못된 값이 입력 되었을 경우, 에러 메시지를 출력하고 그 부분을 다시 입력 받는다.
