# 🚗자동차 경주🚗
## 📃기능 목록📃

### 입력 기능
- [x] 자동차 이름 입력
- [x] 시도할 횟수 입력

### 출력 기능
- [x] 자동차 이름 입력을 위한 출력
- [x] 시도할 횟수 입력을 위한 출력
- [x] 실행 결과 출력
- [x] 우승자 출력
- [x] [ERROR] 메시지 출력
- [x] 에러 이후 재입력

### 자동자 전진, 멈춤
- [x] 무작위 값 생성
- [x] 각 자동차의 무작위 값에 대한 전진, 멈춤 결정

### 우승자 결정
- [x] 게임 완료 후, 누가 우승했는지 결정

### IllegalArgumentException 발생
- [x] 입력 받은 횟수가 숫자가 아닌 경우
- [x] 자동차 이름이 제대로 입력되지 않은 경우 (null 또는, 이름이 6자 이상인 경우)
