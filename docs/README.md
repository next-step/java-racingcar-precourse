# 자동차 경주 게임

## 요구사항

## 목적
- 함수를 분리하는 것에 대한 연습
- 클래스를 분리하는 것에 대한 연습
- 도메인을 풍부하게 만들어보기
- 도메인에 집중한 테스트 코드 작성

## 구현할 기능
- [] 자동차에 대한 이름을 부여하는 입력을 받는다.
  - [] 공백값이 입력되었는지 검증 필요 -> 실패시 IllegalArgumentException/[ERROR] 메시지 출력
  - [] 자동차별 이름이 5자 이하인지 유효성 검증 -> 실패시 IllegalArgumentException/[ERROR] 메시지 출력
  - [] 쉼표(,)를 통해 각 자동차가 구분되는지 확인 되어야 함
- [] 시도 횟수에 대한 입력을 받아 전역적으로 관리되도록 저장한다.
  - [] 1이상의 숫자이며, 정수 바운더리 안이어야한다. 
- [] 자동차 별 전진 혹은 멈춤에 대한 상태를 관리 한다.
  - [] 랜덤한 정수가 4이상이면 전진이며 3이하이면 멈춤 상태
    - [] camp.nextstep.edu.missionutils.Randoms의 pickNumberInRange()를 이용
- [] 각 회차별 실행 결과 출력
  - [] 각 자동차별 전진한 횟수 * '-' 형태로 출력
- [] 우승자 판별 필요
  - [] 자동차 List에서 전진횟수가 가장 많은 자동차를 판별하여 쉼표(,) 구분하여 출력
