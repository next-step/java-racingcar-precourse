# 자동차 경주 게임 기능 요구 사항

<h2>MVC 패턴 사용</h2>

<h3>Model </h3>
<li>Car : 이름 유효성 검증, 랜덤값을 통한 전진 기능</li>
<li>Race : Car의 상태(move) 업데이트, 우승자 찾는 함수</li>

<h3>View </h3>
<li>Input : Car 이름 입력, 시도 횟수 입력(및 유효성 검증)</li>
<li>Output : Car의 move 출력, 우승자 출력</li>

<h3>Controller </h3>
<li>RaceController : Car 생성, 움식일 횟수 유효성 검증 및 가져오기, 게임 시작</li>
