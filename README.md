# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
- 자동차 목록을 입력받는다.
  - 목록의 각 원소의 길이는 1글자 이상, 5글자 이하로 제한한다.
  - 자동차의 이름의 시작과 끝엔 공백이 올 수 없다.
  - 빈값을 입력받으면, 다시 자동차 목록을 입력하도록 사용자에게 알린다.
  - 목록 내의 동일한 자동차 이름이 있다면, 새로 입력 받도록 한다.
- 게임의 라운드 수를 입력받는다.
  - 1 이상의 정수값만 입력받을 수 있다.
  - 그 이외의 값이 들어오면, 다시 입력받도록 사용자에게 알린다.
- 자동차의 이동은 랜덤함수로 결정된다.
  - 자동차는 앞으로만 이동할 수 있다.
  - 0 ~ 9 의 랜덤 값 중 4 이상의 값이 나오면 앞으로 움직이고, 그렇지 않을 땐 움직이지 않는다.
- 우승자는 가장 많이 움직인 자동차를 선정한다.
  - 우승자가 여럿일 경우 쉼표(,)로 구분하여 모두 출력한다.
- 제약 사항 이외의 값을 입력받으면, IllegalArgumentException 을 던진다.
  - "[ERROR]" 문구를 가장 먼저 출력 후 에러의 원인을 설명한다.
