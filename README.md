# 자동차 경주 게임 - 안지원
## 구현 해야할 기능 목록

- 자동차가 움직이는 기능
  - 0~9 사이 ramdom 값을 구한다.
  - 전진 : ramdom 값이 4 이상인 경우
  - 멈춤 : random 값이 3 이하인 경우

- 자동차에 이름을 부여하는 기능
  - 각 자동차에 이름을 부여할 수 있다.
  - 이름은 5자 이하만 가능하다.

- 승리자 판별 기능
  - 가장 많이 이동한 자동차가 승리한다.
  - 우승자가 여러명일 경우 쉼표(,)로 이름을 구분한다.

- 사용자의 입력 기능(1) : 경주할 자동차 이름 입력 기능
  - 자동차 이름을 1개 이상 입력해야 한다.
  - 자동차가 2개 이상일 경우 쉼표(,)로 기준으로 구분해서 입력해야 한다.
  - 자동차 이름은 5자 이하로 입력해야 한다.
  - 사용자가 잘못된 값을 입력한 경우
    - 자동차 이름을 입력하지 않은 경우
    - 자동차가 2개 이상이지만, 쉼표(,)로 구분하지 않은 경우
    - 자동차 이름이 5자 초과인 경우
    - [ERROR]로 시작하는 에러 메시지를 출력하고 게임을 계속 진행할 수 있어야 한다.

- 사용자의 입력 기능(2) : 이동 횟수 입력 기능
  - 0 ~ 2^31 - 1 사이의 값이 입력되어야 한다.
  - 사용자가 잘못된 값을 입력한 경우
    - 0 미만의 값을 입력한 경우
    - 2^31 - 1 초과값을 입력한 경우
    - [ERROR]로 시작하는 에러 메시지를 출력하고 게임을 계속 진행할 수 있어야 한다.

- 입력 메시지 출력 기능
  - 게임 시작 시 자동차 이름 입력 메시지
  - 시도할 횟수 입력 메시지

- 출력 메시지 출력 기능
  - 주어진 횟수 동안의 자동차 이동 현황 메시지
  - 최종 우승자 메시지
