# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 구현할 기능 목록
- [ ] 자동차 생성
  - [ ] 이름을 가지고 있다.
    - 이름은 5자 초과 Exception 처리
  - [ ] 위치값을 가지고 있다.
  - [ ] 엔진을 가지고 있다.
  - [ ] 전진 기능이 있다.
    - 전진 조건은 0~9 사이의 숫자중 4 이상을 받으면 전진한다.
  - [ ] 위치값으로 서로간의 비교가 가능하다.
- [ ] 사용자 입력받기
  - [ ] 사용자는 자동차 이름을 입력할 수 있다.
    - 자동차 이름은 쉼표(,) 기준으로 구분
    - 자동차 이름은 5자 초과시 Exception 처리
    - 입력받은 자동차 이름 숫자만큼 자동차를 생성한다.
  - [ ] 사용자는 몇번의 이동을 할 것인지 입력할 수 있다.
    - 숫자가 아닌 경우 Exception 처리
    - 양수가 아닌 경우 Exception 처리
  - [ ] Exception 처리
    - IllegalArgumentException 발생 후 "[ERROR]" 로 시작하는 메시지 출력
    - 에러 시점 부터 다시 입력 받는다.
- [ ] 출력하기
  - [ ] 이동 횟수만큼 출력한다.
    - 전진하는 자동차 이름을 같이 출력
    - 이동 횟수만큼 "-" 가 늘어난다.
- [ ] 경주 게임
  - [ ] 입력받은 자동차 이름 숫자만큼 자동차를 생성
  - [ ] 입력받은 이동 횟수만큼 자동자 move
    - 이동시 자동차 이름과 위치값을 (이름 : -)로 출력한다
  - [ ] 사용자가 입력한 이동 횟수가 끝나면 종료한다.
    - 누가 우승했는지 알려준다.
    - 한 명 이상일 경우 쉼표(,)로 이름을 구분해 출력한다.
