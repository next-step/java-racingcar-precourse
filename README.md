# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
1. 자동차경주게임 Service를 생성한다. Application에서는 RacingCarGame의 start메소드를 통해 게임을 시작한다.
2. 자동차 이름을 입력받는다. 5자 이하만 가능하며 쉼표를 기준으로 구분한다.
3. 경주 횟수를 입력받는다.
4. 각 자동차는 0~9 사이의 랜덤 값을 구해 4이상인 경우에만 전진한다.
5. 경주 종료 후, 우승한 자동차명을 출력한다. 한 명 이상인 경우 쉼표로 구분한다.
6. 잘못된 값이 입력된 경우 `[ERROR]`로 시작하는 에러메시지를 출력하고 재입력받는다.
7. 도메인 로직에 대한 단위테스트를 구현한다.
