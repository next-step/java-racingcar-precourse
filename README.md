# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 데이터 처리
  + 자동차 생성
    - 이름 검증
    - 전진
  + 자동차 그룹 생성
    - 자동차 추가
    - 자동차 이름 중복 확인
    - 인덱스로 자동차 상태 조회
    - 모든 자동차 이동
    - 우승한 자동차들의 인덱스 조회
  + 자동차 이름 입력값 검증
  + 전진 횟수 입력값 검증
* 화면 출력
  + 자동차 이름 입력 메시지 출력
  + 전진 횟수 입력 메시지 출력
  + 자동차 그룹 전진 결과 메시지 출력
  + 게임 우승자 안내 메시지 출력
  + 에러 메시지 출력
    - 자동차 이름 입력값 에러 메시지 출력
    - 전진 횟수 입력갑 에러 메시지 출력
* 게임 진행
  + 게임 시작
  + 게임 종료
