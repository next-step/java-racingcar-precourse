# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 조건에 따른 진행 결정
    * 조건이 4이상인 경우 - 전진
    * 조건이 3이하인 경우 - 정지
* 랜덤으로 조건 부여(전진, 진행 조건)
* 최종 우승자 판정
* 사용자 입력
    * 자동차 이름 입력(5자 이하만 가능)
    * 레이스 횟수 입력(100회 이하로 제)
* 결과 출력
    * 자동차 위치 출력
    * 우승자 출력
    * 잘못된 값 입력 에러메세지 출력
