# Racing car

## 기능 목록

1. 게임 설정을 한다.
   1. 자동차 이름을 입력으로 받는다.
   2. 라운드 횟수를 입력으로 받는다.
2. 라운드를 진행한다.
   1. 라운드 진행에 따라 자동차 위치를 업데이트한다.
   2. 라운드 종료시 마다 현재 자동차 위치를 화면에 업데이트한다.
3. 결과를 출력한다.
   1. 우승자를 출력한다.

## 객체 역할과 책임

- RacingGameController
  - RoundOrganizer에게 자동차 생성을 위한 메시지를 보낸다.
  - 라운드 회수를 입력으로 받는다.
      - 라운드 횟수 입력을 받는다.
      - 잘못된 입력이 들어온 경우 예외를 던진다.
  - 각 라운드를 진행한다.
  - 참여한 자동차를 관리한다.
- RacingOrganizer
  - 참여하는 자동차를 생성한다.
    - 5 글자 이상인 입력을 받을 수 없다.
    - 잘못된 입력이 들어온 경우 예외를 던진다.
- Car
  - Engine(Random) 결과를 통해 이동할지 결정한다.
  - 이름과 위치 정보를 관리한다.
- QuestionView
  - 게임 설정에 필요한 질문 뷰
- ErrorView
  - 잘못된 입력에 필요한 응답 뷰
- GameBoardView
  - 현재 자동차 위치를 전시한다.
  - 전달 받은 참가자 자동차 목록에서 우승자를 출력한다.
