# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능목록
* 사용자로부터 자동차 이름을 입력받는다.
    - 5자 이하 자동차 이름 검증
    - 자동차 이름은 쉼표(,)를 기준으로 구분한다.
* 사용자로부터 몇 번의 이동을 할 것인지 입력 받는다.
    - 정수 값을 입력받았는지 검증
* 자동차는 0에서 9사이의 Random 값을 통해 전진 또는 멈춘다.
    - random 값이 4이상인 경우 -> 전진
        - 자동차 전진횟수 카운트
    - random 값이 3이하인 경우 -> 멈춤
* 자동차 경주를 시작한다.
    - 각 자동차의 random 값이 4이상인 경우 -> 전진
    - 각 자동차의 random 값이 3이하인 경우 -> 멈춤
* 자동차 경주 완료 후 누가 우승했는지 알려준다.
    - 우승자가 한명 이상일 경우 쉼표(,)로 이름을 구분해 출력한다.
