# 자동차 경주 게임

---------------------------------------
### 자동차 경주 게임 룰
> 게임의 참전 한 n대 자동차들은 입력 받은 시도 횟수 만큼 전진 할 경우 레이싱에서 우승하는 게임이다.
> 
> 전진의 조건은 `4` 이상의 경우 전진하고 `3` 이하의 값이면 멈춘다.
> 

### 자동차 경주 게임 기능 목록
1. [️️✔] 경주 참가 할 자동차 이름을 쉼표(,) 기준으로 입력 받는 기능
  
    1-1. [✔] 자동차 이름은 5자 이하만 입력 가능 하도록 validation 기능
2. [✔] 경주 이동 횟수 입력 받는 기능
   
    2-1. [✔] 경주 이동 횟수는 숫자만 입력 가능하도록 validation 기능 
3. [] 회차 별 참가자들 숫자 뽑기 기능 
4. [] 참가자들 뽑기 기능을 통해 뽑은 숫자 확인 후 전진 및 멈춤 확인 기능 
5. [] 마지막 회차까지 완료 후 최종 우승자 출력 기능
