# java-racingcar-precourse

## Features

### ✏️ Input Validation

잘못된 값을 입력할 경우 에러 메시지를 포함한 `IllegalArgumentException`을 발생시킨 후 다시 입력을 받음

#### 자동차 이름 입력
- 🚫 공백을 포함한 경우
- 🚫 숫자 또는 특수문자를 포함한 경우
- 🚫 이름의 길이가 0이거나 5를 초과하는 경우
- 쉼표로 끝나는 입력값 허용

#### 시도 횟수 입력
- 🚫 정수가 아닌 경우
- 🚫 0 이하의 수인 경우

### 🎮 Game

#### 게임 진행
- 입력 받은 이름의 개수만큼 경주 자동차 생성
- 시도 횟수만큼 게임 진행
  - 각 시도 후 모든 자동차의 전진 횟수 출력
- 게임 완료 후 우승자 판별

#### 전진 여부
- 난수를 생성하여 전진 여부 판단
  - 4 이상일 경우 전진
  - 4 미만일 경우 정지

#### 우승자 판별
- 가장 많이 전진한 자동차가 우승
  - 우승자는 한 명 이상일 수 있음
- 우승자 출력
  - 우승자가 여러 명일 경우 쉼표를 이용하여 구분
