# 자동차 경주 게임

## 기능 요구사항

* 게임 실행을 위한 사용자 입력 값
  - 잘못된 값을 입력한 경우
    - [ERROR]로 시작하는 메시지를 출력
    - 재입력 가능 해야함
  - 자동차에 이름 입력
    - 이름은 쉽표 기준으로 구분
    - 길이는 5자 이하
  - 게임 시도 횟수 입력
    - 숫자만 입력 가능
    - 음수가 될 수 없음
  

* 자동차 게임 실행
    - 입력 된 횟수만큼 전진 또는 정지할 수 있음
    - 랜덤값이 4 이상이면 전진
    - 랜덤값이 3 이하면 정지
    - 텀마다 중간 상태를 출력 할 수 있음
  
  
* 자동차 동작을 위한 랜덤 값
    - 0 ~ 9 사이만 가능
  
  
* 자동차 게임 우승자 판별
  - 게임 종료 시 게임 결과로 부터 우승자 판별
  - 우승자가 여러명인 경우 쉼표로 이름을 구분하여 출력
