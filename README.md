# 자동차 경주 게임
## 기능 목록 정리
- 각 자동차에 이름을 부여할 수 있다.
  - 사용자로부터 입력 받는 이름 예외처리는 다음과 같다.
  - 쉼표(,)를 기준으로 구분하여 이름이 5자 이하인가?
- 사용자로부터 시도할 회수를 입력받는다.
  - 사용자로 부터 입력 받는 회수 예외처리는 다음과 같다.
  - 숫자를 입력했는가?
- 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException를 발생시키고, "[ERROR]"로 시작하는 에러 메시지를
출력 후 그 부분부터 입력을 다시 받는다.
- 자동차 생성
  - 이름, 이동거리 생성

- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
  - 자동차에 전진 함수 생성
  - 전진 멈춤 조건은 다음과 같다. 
  - 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.

- 자동차 리스트 생성
  - 모든 자동차 라운드 결과 함수 생성
  - 우승자 리스트 결과 함수 생성
  
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 실행결과를 알려준다.   
  - 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
  - 우숭자가 한명일 경우 그대로 출력한다.
