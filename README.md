# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* [X] 자동자 전진 기능
* [X] 주어진 횟수 동안 반복 기능
* [X] 자동차 생성시 이름 부여 기능
* [X] 자동차 전진 상태 출력 기능
* [X] 이동 횟수 입력 기능
* [X] 자동차 이름 입력 기능
* [X] 자동차 여러건 생성 기능(쉼표를 기준으로 구분)
* [X] 자동차 이름 5자 이하일 때만 생성 기능
* [X] 0에서 9 사이에서 random 값 구하는 기능
* [X] random 값이 4 이상일 경우 전진, 3 이하일 경우 멈추는 기능
* [X] 게임 완료 후 우승자 확인 기능
* [X] 게임 완료 후 우승자 출력 기능(우승자가 여러 명일 경우, 쉼표로 이름을 구분해 출력)
* [X] 사용자가 잘못된 값을 입력할 경우 [ERROR]로 시작하는 에러 메시지 출력 기능
* [X] 이동 횟수값 포장
* [X] 자동차이름 포장
* [X] 사용자에게 보여지는 기본 메시지 출력 기능
* [X] 자동차 경주 기능
* [X] 다수의 자동차 이동 기능
* [X] 반복 횟수 포장
* [X] 공통 에러 처리 기능
* [] 에러시 재입력 기능 추가
