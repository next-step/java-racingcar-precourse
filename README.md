# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
1. n대 자동차 세팅
   1. 자동차 이름 입력
   1. 자동차 이름 파싱
   1. 자동차 이름 유효성 체크
   1. 자동차 대수 및 이름 세팅
1. 자동차 이동
   1. 자동차 이동 횟수 입력
   1. 자동차 이동을 위한 랜덤 값 생성
   1. 자동차 이동 시키기
      1. 랜덤 값이 4이상 이면 전진
      1. 랜덤 값이 3이하 이면 정지
   1. 자동차 이동 후 진행 과정 출력
1. 게임 종료
   1. 주행거리가 가장 긴 자동차 선정
   1. 게임 우승자 출력
