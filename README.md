# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능구현
* 자동차가 전진이 가능한지 여부를 판단하는 기능 개발
  * 랜덤한 숫자 값이 4이상인지 판단하여 드라이브 상태를 리턴하는 기능 개발

* 자동차를 사용자가 입력한 횟수만큼 전진시키는 기능 개발
  * 드라이브 상태에 따라 포지션을 한칸씩 전진시키는 기능 개발
  * 경쟁자 한명과 이동한 포지션을 비교하여 이겼는지 졌는지 판단하여 승리결과를 리턴하는 기능 개발
  
* 우승자를 판단하는 로직 개발
  * 경쟁자 전부와 비교하여 추월당한적이 한번도 없을경우 승리자로 판단하는 기능 개발
  * 한번도 추월당한적이 없는지 저장하는 결과 클래스 개발
  * 우승자를 추출하는 기능 개발

* 결과를 출력하는 기능 개발
  * 라운드가 진행될때마다 포지션을 출력하는 기능 개발
  * 우승자를 출력하는 기능개발
  
* 사용자로부터 경주 게임과 관련된 값을 입력받는 기능 개발
  * 입력값이 에러일경우 그지점부터 다시시작하는 예외처리 기능 개발
  * 사용자로부터 입력받은 값을 읽어오는 기능 개발
