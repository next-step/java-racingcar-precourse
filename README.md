# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 구현할 기능 목록

- [ ] 자동차의 이름은 쉼표(,)를 기준으로 구분한다.
- [ ] 자동차의 이름은 5자 이하이다.
- [ ] 사용자로부터 몇 번 이동할 것인지 입력받는다.
- [ ] 입력 받은 횟수 동안 n 대의 자동차는 전진 또는 멈춘다.
- [ ] 0 에서 9 사이의 random 값을 구한다.
- [ ] 구한 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
- [ ] 실행 결과를 출력할 때 자동차의 이름을 같이 출력한다.
- [ ] 게임을 완료한 뒤 누가 우승했는지 알려준다. (우승자는 한 명 이상일 수 있다.)
