# 자동차 경주 게임
## 기능 요구사항
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
- 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하면 가능하다.
- 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
- 전진하는 조건은 0에서 9사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한 명 이상일 수 있다.

---

## 세부 기능 요구사항

### 입력 요구사항
- 게임 시작시 '경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)' 텍스트를 출력해서 사용자에게 보여준다.
- 자동차이름은 5자를 초과할 수 없다.
- 자동차 이름을 입력받은 뒤 '시도할 회수는 몇회인가요?'텍스트를 출력해서 사용자에게 보여준다.
- 숫자만 입력받을 수 있다.
- 자동차 이름은 쉼표로 구분되어 생성된다.

### 게임 실행 요구사항 
- 자동차 생성 시 입력받은 문자열에서 쉼표(,)를 기준으로 자동차 생성
- 자동차는 우승을 알기 위해 자동차 별로 전진횟수를 가지고 있어야 한다.
- 0에서 9사이의 난수 발생
- 전진 : 난수가 4이상의 값
- 멈춤 : 난수가 3이하의 값



### 출력 요구사항
- 게임 시작 시 '실행결과' 텍스트 표시
- 실행결과 텍스트 다음줄부터 '[자동차이름]:[-]'텍스트 표시 [-]는 전진횟수만큼 표시(자동차별로 한줄씩 표시)
- 다음게임 시 한줄 공백 후 결과 표시
- '게임 종료 시 '[우승 자동차이름]'가 최종 우승했습니다.' 텍스트 표시([우승 자동차이름]은 쉼표(,)로 우승 자동차이름 모두 표시)


### 프로그래밍 요구사항
- 일급콜렉션을 활용해 구현한다.
- 모든 원시값과 문자열을 포장한다.
