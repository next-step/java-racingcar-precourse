# 자동차 경주 게임

## 기능 목록
- n대의 자동차 이름을 입력 받는다
  - 자동차의 이름의 길이는 5자 이하만 가능(완료)
  - n대의 자동차의 이름은 쉼표(,)를 기준으로 구분
- 몇번의 이동을 할 것인지 입력 받는다
- 사용자 입력 횟수 만큼 자동차들의 이동 여부 판단
  - 0~9 사이의 random 값을 구한다(완료)
  - random 값이 4이상일 경우 전진, 3이하의 경우 제자리(완료)
- 이동 횟수 만큼 결과를 구한다
  - 자동차 여러대에 대해 random 값을 이용해 이동 후 결과 출력(완료)
- 최종 우승자 결과를 구한다
  - 우승자 목록 생성 (완료)
  - 우승 결과 문구 출력
