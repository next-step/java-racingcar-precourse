# 자동차 경주 게임
* 주어진 횟수동안 n대의자동차는 전진 or 멈춤
* 각 자동차에 이름을 부여할수있다. 전진하는자동차를출력할때자동차이름을같이출력한다.
* 자동차이름은 쉼표(,)를기준으로 구분. 이름은5자이하만가능
* 사용자는 몇번의이동을할것인지를 입력할수있어야한다.
* 전진하는조건은 0에서9사이에서 random값을구한후 random값이4이상일경우전진. 3이하의값이면멈춤
* 자동차경주게임을완료한후누가우승했는지를알려준다.우승자는한명이상일수있다.
* Design
    * 자동차
    * 자동차이름
    * 5자이하만가능
    * 자동차들
    * 전진한다
    * 멈춘다
    * 이동값
        * 4이상 전진
        * 3이하 멈춤
    * 우승자
    * View
        * 쉼표로이름구분한다
        * 몇번이동할지받는다
        * 실행결과출력
        * 우승자출력
