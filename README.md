# 자동차 경주 게임

## 요구 기능 정리

### 자동차

- 자동차 생성
    - 지정되는 이름이 꼭 필요
- 자동차의 이름 설정
    - [예외] 자동차의 이름을 5자를 넘을 수 없음
    - [예외] 앞뒤공백을 제거한 길이가 1글자 이상이어야함
- 자동차의 이름 조회
- 자동차의 전진
- 자동차의 전진 여부 결정
    - 입력받은 랜덤값중 4이상의 값이 나오면 전진
    - 입력받은 랜덤값중 3이하의 값이 나오면 정지
- 자동차의 현재 이동 거리 조회
- 자동차의 이름과 이동거리 문자열로 변환

### 자동차들

- 자동차 목록 생성
- 자동차 추가
- 자동차 수 반환
- 전체 자동차 레이싱
- 현재 자동차들 중 최대 이동거리 조회
- 현재 자동차들의 이동거리를 문자열로 변환

### 레이싱

- 자동차 참가 기능
- 이름목록 전달시 차량등록 기능
- 레이싱 진행 기능
- 우승자 선정 기능
- 우승자 문자열로 변환

### 개최팀

- 레이싱 생성
- 레이싱 차량 등록
- 입력 횟수에 따라 레이싱 진행
    - [예외] 입력 횟수는 1번 이상이어야 한다.
- 레이싱 결과 전달
- 우승자 전달

### 레이싱 결과

- 시도마다 레이싱 결과를 추가
- 전체 레이싱 결과를 문자열로 변환

### 승리자들

- 우승 자동차 저장
- 우승자 목록을 문자열로 반환

### 개최팀 (컨트롤러)

- 사용자의 입력을 참가자 목록으로 변환후 참가 등록
- 시도 횟수 입력
- 전체 문자열로 변환

### 게임화면 (뷰)

- 참가자 목록 입력기능 (예외발생시 메세지 출력 후 다시입력)
- 시도 회수 입력기능 (예외발생시 메세지 출력 후 다시입력)
- 레이싱 결과 출력기능
- 우승자 출력 기능

### 입력 유효성 검사

- 사용자의 차량 이름 입력에 관한 유효성 검사
- 사용자의 횟수 입력에 관한 유효성 검사
