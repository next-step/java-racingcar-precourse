# 레이싱 게임

## 개요

레이싱 게임은 정해진 횟수 안에 가장 멀리 이동한 레이싱카를 우승으로 결정하는 게임이다.

## 이력

| 버전 | 업데이트 내역 |
| --- | --- |
| v.1 | 레이싱카 용어사전, 요구사항, 모델링 작성 |

## 용어사전

| 한글명 | 영문명 | 설명 |
| --- | --- | --- |
| 결과화면 | ResultBoard | 레이싱 게임의 결과를 기록한다. |
| 주사위 | Dice | 0이상 9 이하의 숫자가 나온다. |
| 레이싱 | Racing | 주사위의 숫자에 상태를 결정한다. |
| 주행 | Moving | 서킷을 주행한다. |
| 정지 | Stop | 서킷을 주행하지 않는다. |
| 서킷 | Circuit | 레이싱카가 게임을 진행하는 곳 |
| 레이싱카 | RacingCar | Circuit 을 주행하는 차량 |

## 요구사항

### 레이싱게임 UI

### 레이싱카

* 레이싱카의 이름은 규칙에 따라 설정한다.
* 레이싱카는 `주사위` 의 결과에 따라 주행/정지를 반복한다.

### 규칙

* `레이싱카` 의 이름은 1글자 이상, 5글자 이하이다.
* `주사위`의 눈은 0이상, 9이하이다.
* `주사위`의 눈이 0이상 3이하일 경우, `레이싱카`는 `정지` 하고, 4이상 9이하일 경우 `레이싱카`는 `주행` 한다.
* `레이싱카` 가 `주행` 할 경우, 한 칸 이동한다.
* 가장 많이 이동한 `레이싱카` 가 우승이다. 우승이 2명이상인 경우 공동 우승으로 한다.

## 모델링

### 레이싱카

* `레이싱카`는 `이름`, `현재위치`, `주사위` 를 가진다.
* `레이싱카`는 `현재위치` 를 기록한다.
* `레이싱카`는 `서킷`마다 `주사위`를 던져 `주행`/`정지` 을 결정한다.

### 규칙

* `레이싱카`의 이름 제약사항을 정의한다.
* `주사위` 눈금을 정의한다.
* `레이싱카` 의 `주행`/`정지` 상태를 정의한다
* `우승` 조건을 정의한다.
