# 자동차 경주 게임
## 구현할 기능
1. 입력 기능
    * n개의 경주할 자동차 이름을 입력
    * 경주할 횟수 입력


2. 결과 출력
    * 횟수별로 자동차의 전상태 표시
    * 가장많이 전진한 사람이 우승자 출력


3. 요구사항
    * 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하도록 처리
    * 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이
      하의 값이면 멈춘다.


4. 예외사항
    * 자동차 이름이 5자리를 초과할 경우
    * ',' 사이에 문자가 존재하지 않을 경우
    * 중복된 자동차 이름을 입력할 경우
    * 시도 횟수가 자연수가 아닐 경우
