# java-racingcar-precourse

## 구현 기능
- [ ] 경주할 자동차 정보(이름) 입력
  - [ ] 쉼표(,)로 구분
  - [ ] 5자 이하만 가능
  - [ ] 1대 이상의 자동차 입력
- [ ] 시도 횟수 입력
  - [ ] 숫자만 입력 가능
  - [ ] 1회 이상 시도 가능
- [ ] 자동차 경주 게임 구현
  - [ ] 전진 조건은 0~9 사이의 무작위 수 중 4 이상인 수
  - [ ] 주어진 시도 횟수 만큼 실행 결과 출력
- [ ] 우승자 출력
  - [ ] 우승자가 여러명인 경우 쉼표(,)로 구분
  - [ ] 우승자는 1명 이상

---

## 클래스 정보
- Car
  - 자동차 이름
  - 현재 위치
- RacingGame
  - 자동차 경주 준비
  - 자동차 경주 실행
  - 우승자 출력
- Input
  - 경주할 자동차 정보 입력
  - 시도 횟수 입력
- Output
  - 실행 결과 출력
  - 우승자 정보 출력
