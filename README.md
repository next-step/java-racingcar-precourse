# 자동차 경주 게임

## 도메인
- Car
  - `Car`는 `CarName`을 가진다
  - `Car`는 `Distance`를 가진다
- CarName
  - `Car`는 `이름`을 가진다
  - `이름`은 5자 이하만 가능하다
  - `이름`은 null 혹은 빈값이 아니다
- Distance
  - `Car`는 `이동 거리`를 가진다
- Cars
  - `Cars`는 `자동차 리스트`를 가진다
- Race
  - `Race`는 `자동차 리스트`를 가진다
- MoveResult
  - `MoveResult`는 `이름`과 `이동 거리`를 가진다
- RaceResult
  - `RaceResult`는 `우승자 이름`을 가진다

## 요구사항
- 사용자는 참가할 자동차 이름을 입력한다
  - 자동차 이름은 쉼표로 구분
- 사용자는 이동 횟수 입력한다
- 자동차는 이름을 가진다
  - 자동차 이름은 5자 이하만 가능
- 자동차는 전진 또는 정지한다
  - 전진 조건은 4 이상
  - 정지 조건은 3 이하
- 경주를 완료하면 우승자를 출력한다
  - 우승자가 한명이면 쉼표(`,`)로 구분
- 사용자가 잘못 입력한 경우 "[ERROR]"로 시작하는 에러 메세지를 출력 후 다시 입력받는다


## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
