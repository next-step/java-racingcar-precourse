# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 클래스

- `car`
  - `engine`
    - `Engine` : 엔진. 임의의 출력을 내고, 전진할지 멈출지 판단
    - `Power` : 출력
  - `position`
    - `Position` : 자동차 위치
  - `Car` : 자동차
  - `CarName` : 자동차 이름
  - `CarNames` : 사용자 입력을 받아 자동차 이름들로 변환
  - `CarRecord` : 자동차 기록 (자동차 이름, 당시의 위치 저장)
  - `Cars` : 자동차 목록 일급 컬렉션
  - `CarStatus` : 자동차 상태 (전진, 멈춤)
- `race`
  - `Race` : 경주
  - `RaceRecord` : 경주 기록 (라운드 기록 저장)
  - `Round` : 라운드 (시도할 횟수)
  - `RoundRecord` : 라운드 기록 (자동차 기록 저장) 
- `utils`
  - `StringUtils` : 문자열 유틸성 클래스
- `Console` : 메세지 출력
- `Main` : 사용자 입력을 받고 경주를 실행
 

## 기능 목록

- [x] 사용자 입력 받기
  - [x] 입력받기 전 메세지를 출력한다.
  - [x] 사용자로부터 자동차 이름 및 시도 횟수를 입력받는다.
- [x] 경주 준비
  - [x] 자동차의 이름은 쉼표(,)를 기준으로 구분한다.
  - [x] 자동차의 이름은 5자 이하이다.
  - [x] 자동차의 이름이 6자 이상일 경우 예외를 던진다.
  - [x] 자동차의 처음 위치는 0 이다.
  - [x] 시도 횟수를 라운드 수로 설정한다.
- [x] 경주 시작
  - [x] 라운드 수 만큼 라운드를 실행한다.
  - [x] 라운드가 끝나면 라운드 기록을 경주 기록으로 저장한다.
- [x] 라운드 시작
  - [x] n 대의 차는 주행(전진 또는 멈춤한다.)
  - [x] 주행 후 자동차 기록을 라운드 기록으로 저장한다.
- [x] 자동차 주행
  - [x] 엔진이 0 에서 9 사이의 임의의 출력을 낸다.
  - [x] 출력이 4 이상일 경우 전진, 3 이하일 경우 멈춘다.
  - [x] 전진하면 위치가 1 만큼 증가하고, 멈추면 위치가 그대로 유지된다.
- [x] 경주 완료
  - [x] 저장했던 경주 기록을 바탕으로 실행 결과를 출력한다.
  - [x] 실행 결과를 출력할 때 자동차의 이름을 같이 출력한다.
  - [x] 누가 우승했는지 출력한다. (우승자는 한 명 이상일 수 있다.)
