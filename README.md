# 자동차 경주 게임

## 기능 목록
- n대의 자동차 이름을 입력 받는다
  - 자동차의 이름의 길이는 5자 이하만 가능
  - n대의 자동차의 이름은 쉼표(,)를 기준으로 구분
- 몇번의 이동을 할 것인지 입력 받는다
- 사용자 입력 횟수 만큼 자동차들의 이동 여부 판단가
  - 0~9 사이의 random 값을 구한다
  - random 값이 4이상일 경우 전진, 3이하의 경우 제자리
- 레이싱 게임을 실행한다
  - 자동차 이름 과, 이동회수를 입력 받아 레이싱 게임을 실행
  - 우승자를 구해 화면 에 출력
- 최종 우승자 결과를 구한다
  - 자동차 중 가장 멀리 이동한 자동차의 거리와 같은 거리를 이동한 자동차를 우승자 목록을 만든다
