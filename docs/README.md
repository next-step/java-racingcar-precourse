## 기능 목록
- [x] 자동차 이름 입력
  - 쉼표로 구분
  - 5자 이하만 가능
  - 예외 처리
    - 5자 이하인가?
    - 중복된 이름이 있는가?
    - 공백으로만 입력되었는가?
- [x] 이동 횟수 입력
  - 예외 처리
    - 숫자인가?
    - 0 이상인가?
- [x] 전진
  - 조건 : random 값이 4 이상일 경우 전진, 3 이하면 멈춤
- [ ] 차수별 실행 결과 출력 
- [ ] 우승자 안내
  - 한 명 이상일 경우 쉼표로 구분
- [ ] 잘못된 입력에 대한 메시지 처리
  - 예 : [ERROR] 메시지 내용

## 프로그램 실행 예제
```
경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
pobi,crong,honux
시도할 회수는 몇회인가요?
5
실행 결과
pobi : -
crong :
honux : -

pobi : --
crong : -
honux : --

pobi : ---
crong : --
honux : ---

pobi : ----
crong : ---
honux : ----

pobi : -----
crong : ----
honux : -----

최종 우승자: pobi, honux
```
