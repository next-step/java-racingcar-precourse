# 기능 목록
- 자동차 1회 이동
  - 0에서 9사이의 random 값 1개를 구한다.
  - random 값이 4 이상일 경우 전진한다.
  - random 값이 3 이하일 경우 멈춘다.
- 사용자로부터 입력 받는 이동 횟수만큼 자동차 이동
  - 0보다 작거나 9999보다 크면 예외 처리
  - 자동차 n회 이동
- 사용자로부터 입력 받는 자동차 이름 예외 처리
  - 쉼표(,)를 기준으로 구분
  - 이름은 1~5자리만 가능
  - 이름의 앞뒤 공백 제거
  - "", " ", " ,car1", "car1, ", " , " 등 잘못된 입력 예외 처리
- 경주 진행하고 최종 우승자 판정
  - 가장 많은 횟수를 이동한 자동차가 우승자
  - 공동 우승자인 경우 쉼표와 띄어쓰기로 구분
    - ex) a, b
- 사용자가 입력한 값으로 게임 진행하고 결과 출력
  - 사용자가 잘못된 값을 입력할 경우 에러 메시지를 출력하고 다시 입력을 받는다.
