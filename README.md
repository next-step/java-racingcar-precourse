## 기능 요구사항
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
- 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
- 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
- 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다.
- 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
- 사용자가 잘못된 값을 입력할 경우 “[ERROR]”로 시작하는 에러 메시지를 출력 후 입력을 다시 받는다.

## 구현 기능 목록
### UserInputView
- 사용자가 입력하는 콘솔 입력 함수를 구현한다.

### OuputView
- 결과 등을 출력하는 함수 및 출력 스트링을 정의한다.

### GameRule
- 프로젝트 전체에서 자주 쓰이는 리터럴값을 정의한다.

### Car (Model)
- carName과 distance라는 멤버변수를 가지고 있다.
- 자동차 이름이 제약사항에 위배되는 경우 에러를 throw한다.
   - 제약사항은 1글자 이상 5글자 이하이다.
- 전진, 중지를 결정하는 함수를 만든다. (decideMovement)
- 전진하는 함수를 만든다.

### CarGroup (Model)
- List<Car> 멤버변수를 가지고 있다.
- 자동차 이름이 위배되는 경우 에러를 throw한다.
- 해당 그룹 결과를 출력하는 함수를 만든다.

### CarNames (Model)
- 유저에게 입력받은 String 값을 쉼표 기준으로 구분하여 리스트로 가지고 있다.
- 유효한 Car 이름들인지 체크하는 함수를 만든다.

### Round (Model)
- 초기화는 String, 멤버변수는 int값이고 치환되지 않는다면 에러를 리턴한다.

### Winner (Model)
- 게임에서 승리한 CarGroup을 가지고 있다. (게임의 승리자는 여러명일 수 있기 때문)

### RacingService
- 차들이 움직임을 결정하고, 그 결과를 출력하는 startRacing 함수를 구현한다.

### GameController
- 구현한 View와 Model을 사용하여 기능 구현
