# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 요구사항

- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- 각 자동차에 이름을 부여할 수 있다.
- 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
- 자동차 이름은 쉼표(,)를 기준으로 구분한다
  - 이름은 5자 이하만 가능하다.
- 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
  - 횟수는 숫자만 입력할 수 있다.
  - 0은 입력할 수 없다.
- 전진하값는 조건은 0에서 9 사이에서 random 을 구한다.
  - radnom 값이 4 이상일 경우 전진한다.
  - random 값이 3 이하일 경우 멈춘다.
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다.
- 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
- 사용자가 잘못된 값을 입력할 경우 "[ERROR]"로 시작하는 에러메시지를 출력 후 입력을 다시 받는다.

## 기능 목록

- 자동차
  - n대의 자동차를 생성할 수 있다.
    - 일급 컬렉션 생성
  - 자동차는 전진 또는 멈출 수 있다.
    - 랜덤 값을 구한다. (0 ~ 9)
    - 랜덤 값이 4이상이면 전진한다.
    - 랜덤 값이 3이하면 멈춘다.
  - 자동차에 이름을 부여할 수 있다.
    - 5자 이하의 이름만 가능하다.
    - 빈 값이 올 수 없다.

- 자동차 이름 입력
  - 자동차 이름을 입력한다.
  - 자동차 이름을 (,)로 구분한다.
  - 자동차 이름은 빈 값이 올 수 없다.

- 시도 회수 입력
  - 숫자만 올 수 있다.
  - 0은 올 수 없다.
  
- 출력
  - 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
  - 한 턴마다 모든 차들의 결과를 출력한다.
  - 모든 턴이 끝나면 최종 우승자를 출력한다.
    - 우승자가 여러 명이라면 (,)로 분리한다.

- 에러
  - 사용자가 잘못된 값을 입력할 경우 "[ERROR]"로 싲가하는 에러메시지를 출력하고 입력을 다시 받는다.
