# 자동차 경주 게임



* 사용자는 쉼표로 구분된 n대의 자동차 이름을 입력한다.   
   * 잘못된 값을 입력한 경우 IllegalException이 발생하고 "[ERROR]"로 시작하는 에러 메시지를 출력하고 다시 입력을 받는다.


* 사용자는 자동차가 이동할 횟수를 입력한다.   
    * 잘못된 값을 입력한 경우 IllegalException이 발생하고 "[ERROR]"로 시작하는 에러 메시지를 출력하고 다시 입력을 받는다.


* 입력한 횟수만큼 자동차를 각각 random한 수를 구해 알맞게 이동시킨다.   
   * 구해진 수가 4이상일 경우 전진한다.   


* 누가 우승했는지 출력한다.   
   * 우승자가 한명 이상인 경우 쉼표로 이름을 구분해 출력한다.
