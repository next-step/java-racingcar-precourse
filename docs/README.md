# 자동차 경주 게임

## 요구사항
- 주어진 횟수동안 n대의 자동차는 전진 또는 멈출수있다.
- 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할때 자동차 이름을 같이 출력한다.
- 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자이하만 가능하다.
- 사용자는 몇번의 이동을 할것인지를입력할수있어야한다.
- 전진하는 조건은 0에서9 사이에서 random값을 구한 후 random값이 4이상일 경우 전진하고,3이하의 값이면멈춘다.
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 
- 우승자가 한명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
- 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException를 발생시키고,"[ERROR]"로 시작하는 에러메시지를 출력 후 그 부분부터 입력을 다시받는다.
- Exception이 아닌 IllegalArgumentException, IllegalStateException 등과 같은 명확한 유형을 처리한다.


## 목적
- 함수를 분리하는 것에 대한 연습
- 클래스를 분리하는 것에 대한 연습
- 도메인을 풍부하게 만들어보기
- 도메인에 집중한 테스트 코드 작성

## 구현할 기능
- [x] 자동차에 대한 이름을 부여하는 입력을 받는다.
  - [x] 공백값이 입력되었는지 검증 필요 -> 실패시 IllegalArgumentException/[ERROR] 메시지 출력
  - [x] 자동차별 이름이 5자 이하인지 유효성 검증 -> 실패시 IllegalArgumentException/[ERROR] 메시지 출력
  - [x] 쉼표(,)를 통해 각 자동차가 구분되는지 확인 되어야 함
- [x] 시도 횟수에 대한 입력을 받아 전역적으로 관리되도록 저장한다.
  - [x] 1이상의 숫자이며, 정수 바운더리 안이어야한다. 
- [x] 자동차 별 전진 혹은 멈춤에 대한 상태를 관리 한다.
  - [x] 랜덤한 정수가 4이상이면 전진이며 3이하이면 멈춤 상태
    - [x] camp.nextstep.edu.missionutils.Randoms의 pickNumberInRange()를 이용
- [x] 각 회차별 실행 결과 출력
  - [x] 각 자동차별 전진한 횟수 * '-' 형태로 출력
- [x] 우승자 판별 필요
  - [x] 자동차 List에서 전진횟수가 가장 많은 자동차를 판별하여 쉼표(,) 구분하여 출력
