# 자동차 경주 게임🚕

우아한테크캠프 PRO 프리코스 두 번째 과제인 자동차 경주 게임을 위한 프로젝트이다. 기능 요구사항, 프로그래밍 요구사항, 과제 진행 요구사항이 주어지며 해당 요구사항을
파악하여 기능목록을 작성하고 정리된 기능과 요구사항을 최대한 충족하기 위해 진행 해야하며 주어진 기간내에 제출하는 것을 목표로 한다. 
선발 목적도 좋지만 미션 수행을 위해 투자 한 시간들이 성장의 밑거름이 되도록 의미있는 시간을 보내기 위해 노력 한다.

```markdown
경주할자동차이름을입력하세요.(이름은쉼표(,)기준으로구분)
pobi,crong,honux
시도할회수는몇회인가요?
5

실행결과
pobi:-
crong:
honux:-

pobi:--
crong:-
honux:--

pobi:---
crong:--
honux:---

pobi:----
crong:---
honux:----

pobi:-----
crong:----
honux:-----

최종우승자는pobi,honux입니다.
```
### 기능요구사항
```text
• 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
• 각 자동차에 이름을 부여 할 수 있다. 전진하는 자동차를 출력 할 때 자동차 이름을 같이 출력한다.
• 자동차이름은 쉼표(,)를 기준으로 구분하며 이름은 5자이하만 가능하다.
• 사용자는 몇번의 이동을 할 것 인지를 입력 할 수 있어야한다.
• 전진하는 조건은 0에서 9사이에서 random값을 구한후 random값이 4 이상 일경우 전진하고,3 이하의 값이면 멈춘다.
• 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다.
• 우승자가 한 명이상일 경우,쉼표(,)로 이름을 구분해 출력한다.
• 사용자가 잘못된 값을 입력 할 경우 “[ERROR]”로 시작하는 에러메시지를 출력 후 입력을 다시받는다.
```
### 프로그래밍 요구사항
```text
• 게임을 시작하는 시점은 Application#main()이다.
• JDK 8 버전에서 실행가능해야 한다.
•  nextstep.utils의 Randoms#pickNumberInRange(), Console#readLine()를 활용한다.
• 프로그램 구현을 완료 했을 때 ApplicationTest에 있는 Test Case 2개가 성공해야 한다.
    - 제공하는 소스코드를 참고해 게임을 위한 모든 Test Case를 추가하면 좋은 연습이 될 수 있다.
• 자바 코드 컨벤션을 지키면서 프로그래밍 한다.
    - https://naver.github.io/hackday-conventions-java
• indent는 한 단계 만 허용 한다.
• 자바 8에 추가된 stream api를 사용하지 않고 구현한다. 람다는 가능하다.
• else 예약어를 사용하지 않는다.
• 메소드가 한 가지 일만 잘 하도록 구현하여 메소드의 길이가 10라인을 넘기지 않도록 한다.
• 핵심 로직을 구현하는 코드와 UI를 담당하는 로직을 분리한다.
   - MVC 패턴 기반으로 구현 후 Model에 대한 단위 테스트 추가에 집중
• 학습테스트를 통해 JUnit 학습하기.pdf 참고 학습 후 Junit5기반 단위 테스트 구현 한다.
• 일급 콜렉션을 활용 하여 구현하고, 모든 원시값과 문자열은 포장한다.
```

## 기능 목록 📝
* 게임 실행을 위한 사용자 입력 값
  - 잘못된 값을 입력한 경우
    - [ERROR]로 시작하는 메시지를 출력
    - 재입력 가능 해야함
  - 자동차에 이름 입력
    - 이름은 쉽표 기준으로 구분
    - 길이는 5자 이하
    - 공백일 수 없음
    - 중복일 수 없음
  - 게임 시도 횟수 입력
    - 숫자만 입력 가능
    - 음수가 될 수 없음
    - 공백일 수 없음
  

* 자동차 게임 실행
    - 입력 된 횟수만큼 전진 또는 정지할 수 있음
    - 랜덤값이 4 이상이면 전진
    - 랜덤값이 3 이하면 정지
    - 텀마다 중간 상태를 출력 할 수 있음
  
  
* 자동차 동작을 위한 랜덤 값
    - 0 ~ 9 사이만 가능
  
  
* 자동차 게임 우승자 판별
  - 게임 종료 시 게임 결과로 부터 우승자 판별
  - 우승자가 여러명인 경우 쉼표로 이름을 구분하여 출력

  - Console 기능 검증 테스트
