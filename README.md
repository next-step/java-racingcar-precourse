# 🚘 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현 목록

* 자동차들의 이름를 입력받는다.
  ```
  자동차의 이름은 (,)로 구분한다.
  이름은 다섯자 이하이다.
  공백이 있으면 제거하고 이름 저장한다.
  null이 아니도록 유의한다.
  자동차의 이름들을 입력받으면 각 자동차 객체를 생성한다.
  ```
  
* 움직이는 횟수를 입력받는다.
  ```
  횟수가 숫자가 아니거나 0인 경우 다시 입력받는다.
  움직이는 횟수에 맞추어 각 자동차에게 0~9의 랜덤한 숫자를 생성한다.
  이때 랜덤한 숫자가 4이상이면 전진을 하고 3이하이면 멈춘다.
  ```
  
* 자동차 경주 계임에서 누가 우승했는지 알려준다.
  ```
  우승자가 한 명 이상인 경우 쉼표(,)로 이름 구분 후 출력한다.
  ``` 
  
* 입력 값이 잘못 되면 "[ERROR]"로 시작하는 메시지를 남기고 다시 입력받는다.
