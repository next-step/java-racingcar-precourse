# 자동차 경주 게임
## 기능요구사항
* 주어진 횟수 동안 n대의 전진
* 각 자동차에 이름
  * 이름은 쉼표로 구분
  * 이름은 5자 이하만
* 사용자가 이동 횟수 입력
* 전진 조건 
  * 숫자 범위 : 0 ~ 9
  * 정지 : 난수가 3 이하
  * 전진 : 난수가 4 이상
* 실행 결과
  * 실행 결과는 매 라운드
  * 출력 포맷 : "이름" : "-" (전진 횟수)
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
