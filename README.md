# 자동차 경주 게임

## 요구 기능 정리 

### 자동차
- 자동차자 생성
- 자동차의 이름 설정
  - [예외] 자동차의 이름을 5자를 넘을 수 없음
  - [예외] 자동차의 이름은 공백이 들어 갈 수 없음
- 자동차의 전진 또는 정지 
- 자동차의 현재 이동 거리 조회
- 자동차의 이름 조회

### 레이싱
- 생성된 자동자 저장
  - 자동차의 이름이 지정되어야 한다.
  - [예외] 자동차의 이름은 5자를 넘을 수 없다.
- 자동차의 전진 여부 결정
  - 생성된 0~9사이의 랜덤값중 4이상과 이하를 판단
- 게임진행
- 우승자 선정
- 우승자 표시

### 레이싱 보드 
- 첫 메세지 출력
- 시도 횟수 입력
- 실행 결과 출력
- 우승자 출력

### 공통 
- 잘못된 값 입력시 [ERROR] 로 표시되는 메세지 출력

### 입력 유효성 검사
- 사용자의 횟수 입력에 관한 유효성 검사
