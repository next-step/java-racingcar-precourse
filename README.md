# 자동차 경주 게임
## 기능목록
1. 자동차 사용자 입력(자동차이름, 반복횟수)함수
	- 자동차이름 입력값 검증(5자이하 및 구분자는 ",")
	- 반복횟수 입력값 검증(숫자만)
2. 자동차정보를 저장하는 클래스
	- 속성 = 이름, 전진횟수
            - 메소드 = getter, setter 및 우승자 비교를위한 Comparable 인터페이스구현
3. 자동차 움직임을 담당하는 함수
	- 값이 4이상이면 전진, 3이하면 정지
4. 자동차 주행거리 표시 출력 함수
	- 숫자값만큼 '-' 문자열로 변환해주는 함수
5. 우승후보 출력 함수
	- 입력된 자동차이름 순서대로 우승자를 출력
