# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


## 기능 목록 정리
- 자동차 이름 입력 기능 구현
  - 이름 길이는 5글자 이하여야 한다
- 자동차 경주 시도 횟수 입력 기능 구현
- 자동차 경주 실행
- 자동차 이동 기능 구현
  - 난수 생성 기능 구현
  - 난수 값에 따라 이동 (4 이상일 경우 전진, 3 이하는 정지)
  - 자동차는 이름과 이동거리(또는 위치) 값을 가진다
- 자동차 경주 1회 실행결과 출력기능 구현
- 최종 우승자 선정 기능 구현
- 최종 우승자 출력 기능 구현
