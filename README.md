# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 요구사항 정리
### 자동차
* 자동차는 전진하거나 멈출 수 있다.
  * 각 자동차마다 랜덤한 값을 생성한다 (0 ~ 9) 
  * 0 ~ 9사이에 랜덤값을 구한 후 랜덤값이 4이상이면 전진 3이하이면 멈춘다.
* 자동차는 이름이 있으며 5글자 이하만 가능 (빈값은 x)
* 자동차는 n대 존재하며, 입력시에 이름은 쉼표(,)를 기준으로 구분한다.
* 자동차는 자신이 이동한 거리를 알고있다.


### 게임
* 사용자는 자동차가 이동할 횟수를 입력한다.
* 입력한 횟수만큼 이동을 시도 한 후 가장 많이 이동한 자동차가 우승한다.
* 우승자가 한 명 이상일 경우 쉼표(,)로 이름을 구분해 출력한다.
* 사용자가 잘못된 값을 입력할 경우 "[ERROR]" 로 시작하는 에러메세지를 출력 후 입력을 다시 받는다.
