# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 기능
* 주어진 횟수 동안 n대의 자동차는 전진 또는 정지한다.
    * 값이 4 이상일 경우 전진, 3이하의 값일 경우 정지한다.
    * 횟수 동안 전진 또는 정지한다.
* 전진하는 자동차를 출력 시 자동차 이름이 같이 출력된다.
    * 자동차 이름을 출력한다
    * 1회 전진 시 (-)로 나타낸다.
    * 형식) 자동차명 : ---
* 지동차 경주 게임 완료 후 우승자를 출력한다.
* 사용자는 몇번의 이동을 할 것인지 입력할 수 있다.
    * 입력값이 잘못된 경우 '[ERROR]'로 시작하는 에러 메시지를 출력 후 재입력 받는다.
* 자동차에 이름을 부여한다.
    * 이름은 5자 이하이다
    * 자동차 이름은 쉼표를 기준으로 구분한다.
    * 입력값이 잘못된 경우 '[ERROR]'로 시작하는 에러 메시지를 출력 후 재입력 받는다.
* 0에서 9 사이의 랜덤 값을 구한다.




# 기능 상세
* 이동 조건
    * 4이상 전진
    * 3이하 정지

* 자동차
    * 전진 또는 정지
    * 5자 이하의 이름
    * 전진 횟수
    * 자동차명:전진상황 출력

* 자동차들
    * 자동차에 이름 할당
    * 우승자 찾기
    * 우승자 출력
    * 우승자 없을 시 처리

* 플레이어
    * 시도할 회수를 입력한다.
    * 자동차 이름을 , 기준으로 입력한다.
    * 공백 제거
    * 5자 초과 시 에러 메시지 출력 후 재입력

* 0~9 사이의 랜덤값 생성
