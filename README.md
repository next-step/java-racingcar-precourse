# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

---
## 기능 목록
### 게임 참가자
- 쉼표로 구분되어진 n대의 자동차
- 이름은 5글자 이하
- 몇번의 이동을 할 것인지 입력

### 게임 룰
- 0 ~ 9 사이의 랜덤 값 중 4이상일 경우 전진. 3이하는 멈춤
- 가장 많이 이동한 자동차가 우승
- 우승자는 한 명 이상(공동 우승)

### 자동차
- 이름과 위치값을 가짐
- 게임 룰에 따라 위치값 변경
