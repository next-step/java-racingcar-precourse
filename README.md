
# 자동차 경주 게임
* 이 프로그램은 자동차 경주 게임입니다. 자동차 이름들을 ","로 구분하여 입력한 후 몇 번 스텝을 이동할지 압력합니다.

# 구현 기능 목록

## 자동차 구현
* 자동차 이름 클래스
  * 이름 길이 확인
  * 자동차 이름 얻기
* 자동차 진행 사항 클래스
  * 진행사항 추가
  * 진행사항 얻기
* 자동차 클래스
  * 자동차 정보 얻는 기능
  * 다음 스텝 진행 기능
## 자동차 리스트 클래스
* 자동차 진행 하는 기능
* 자동차 리스트 출력하는 기능
## 임의의 수 생성 기능 구현
* 0-9 까지 숫자 생성 기능
## 사용자 입력을 받고 프린트 하는 기능
* 자동치 이름 목록 받는 기능
* 시도하는 수 입력 받는 기능
## 우승자 확인 클래스
* 우승자 진행 횟수 구하기
* 우승자 이름 얻기
## 자동차 게임 관리
* 자동차 생성
* 자동차 진행 수행
