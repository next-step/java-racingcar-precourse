# 자동차 경주 게임

## 요구 기능 정리

### 입력

- 자동차 이름 입력
- 시도 횟수 입력

### 동작

- 입력한 자동차 이름에 따라 레이싱 참가 자동차 생성
- 시도 횟수에 따라 레이싱 진행
- 참가한 자동차들이 각각 레이싱 진행
- 1~9랜덤한 숫자가 입력되어 4이상의 숫자가 나오면 자동차가 전진
- 우승자 선정

### 출력

- 자동차 입력 안내 메세지 출력
- 시도 회수 입력 안내 메세지 출력
- 시도 횟수마다 전진한 이력 출력
- 우승자 출력

### 예외

- 자동차 이름 입력이 공백인 경우
- 자동차 이름이 한글,숫자,영어가 아닌 경우
- 자동차 이름이 5자가 넘는 경우
- 입력된 이름이 이름1,이름2 형식에 맞지 않는 경우
- 자동차 이름이 중복되는 경우
- 시도 회수가 숫자가 아닌 경우

## 객체별 상세 기능

### 자동차 Car

- 자동차 생성
    - 생성시 지정되는 이름이 꼭 필요
- 자동차의 이름 설정
    - [예외] 자동차의 이름을 5자를 넘는 경우
    - [예외] 앞뒤공백을 제거한 길이가 1글자 이상이 아닌경우
- 자동차의 이름 조회
- 자동차의 전진
- 자동차의 전진 여부 결정
    - 입력받은 랜덤값중 4이상의 값이 나오면 전진
    - 입력받은 랜덤값중 3이하의 값이 나오면 정지
- 자동차의 현재 이동 거리 조회
- 자동차의 이름과 이동거리 문자열로 변환

### 자동차이름들

- 자동차 이름들 생성
- 유효성 검사
    - [예외] 완성된 한글, 숫자, 알파벳만 가능하며 1자이상 5자 이하
    - [예외] 중복되는 이름 불가
- 리스트 형태로 반환

### 자동차들 Cars

- 자동차 목록 생성
- 자동차 추가
- 자동차 수 반환
- 전체 자동차 레이싱
- 현재 자동차들 중 최대 이동거리 조회
- 최대 이동거리인 자동차를 반환
- 현재 자동차들의 이동거리를 문자열로 변환

### 레이싱 Racing

- 자동차 참가 기능
- 자동차 이름들로 자동차 참가
- 레이싱 기능
- 레이싱 시도 횟수 설정
- 시도 회수 만큼 레이싱 진행
- 레이싱 결과 반환

### 레이싱 결과 RacingResult

- 시도마다 레이싱 결과를 추가
- 전체 레이싱 결과를 문자열로 변환
-

### 시도 회수 TryCount

- 입력 받은 시도 횟수를 저장
- 입력받은 문자열 유효성 검사
    - [예외] 입력된 내용이 숫자가 아닌 경우
    - [예외] 숫자가 양수가 아닌 경우

### 승리자들 Winners

- 우승 자동차 저장
- 우승자 목록을 문자열로 반환

### 개최팀 (컨트롤러) Host

- 사용자의 입력을 참가자 목록으로 변환후 참가 등록
- 시도 횟수 입력
- 문자열로 변환
- 입력 사항 유효성 검사

### 게임화면 (뷰) GameView

- 참가자 목록 입력기능 (예외 발생시 메세지 출력 후 다시입력)
- 시도 회수 입력기능 (예외 발생시 메세지 출력 후 다시입력)
- 레이싱 결과 출력기능
- 우승자 출력 기능

### 입력 유효성 검사 Validator

- 사용자의 자동차 입력에 관한 유효성 검사
- 자동차 이름 중복 검사
- 사용자의 횟수 입력에 관한 유효성 검사
