# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 게임 시작

* 자동차 이름 입력 받아서 부여하기
  * 자동차 이름은 쉼표(,)를 기준으로 구분
  * 자동차 이름은 5자 이하만 가능

* 사용자로부터 횟수 입력받기
  * 횟수가 1이상의 숫자인가?

* 자동차 전진 또는 멈춤 기능
  * 0에서 9사이 random 값 중에서, random 값이 4 이상일 경우 전진
  * 0에서 9사이 random 값 중에서, random 값이 3 이하일 경우 멈춤

* 게임 우승자 결과 반환
  * 우승자는 한 명 이상일 수 있음
