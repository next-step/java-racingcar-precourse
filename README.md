# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
- 사용자 입력
    - 자동차 이름은 1자 이상, 5자 이하의 문자열만 가능하다
    - 자동차 이름은 쉼표(,)로 구분하여 N개 만큼 입력받는다
    - 시도할 횟수는 숫자만 허용된다
    - 잘못된 값을 입력할 경우 "[ERROR]" 로 시작하는 에러 메세지를 출력 후 입력을 다시 받는다
- 자동차는 이동할 수 있다
    - random 값을 구한 후 전진 또는 멈춤을 결정한다
      - 0~3 : 멈춤
      - 4~9 : 전진
- 우승자를 선정한다
    - 우승자 이름을 출력한다
    - 우승자가 한 명 이상일 경우 쉼표(,)로 이름을 구분한다
