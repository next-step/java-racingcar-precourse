# 자동차 경주 게임

## 요구 기능 정리

### 자동차

- 자동차자 생성
- 자동차의 이름 설정
    - [예외] 자동차의 이름을 5자를 넘을 수 없음
    - [예외] 자동차의 이름은 공백이 들어 갈 수 없음
- 자동차의 전진 또는 정지
- 자동차의 현재 이동 거리 조회
- 자동차의 이름 조회

### 레이싱

- 생성된 참가자 자동자 저장
    - 자동차의 이름이 지정되어야 한다.
    - 참가자 목록에 추가
- 자동차의 전진 여부 결정
    - 입력받은 랜덤값중 4이상의 값이 나오면 전진
    - 입력받은 랜덤값중 3이하의 값이 나오면 정지
    - [예외] 입력된 값은 0~9 사이의 값이어여 한다.
- 모든 자동차 레이싱 기능
    - [예외] 레이싱 가능한 차량이 없다면 예외 발생
- 가장 많이 이동한 거리 조회
- 우승자 선정
- 우승자 표시

### 개최팀

- 레이싱 생성
- 레이싱 차량 등록
- 입력 횟수에 따라 레이싱 진행
    - [예외] 입력 횟수는 1번 이상이어야 한다.
- 레이싱 결과 전달
- 우승자 전달

### 레이싱 보드

- 첫 메세지 출력
- 이름 입력
    - [예외] 여러명을 입력할 땐 쉼표로 구분해서 입력
- 시도 횟수 입력
    - [예외] 숫자로 된 값만 입력가능
- 실행 결과 출력
- 우승자 출력

### 공통

- 잘못된 값 입력시 [ERROR] 로 표시되는 메세지 출력

### 입력 유효성 검사

- 사용자의 횟수 입력에 관한 유효성 검사
