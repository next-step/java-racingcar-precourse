# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 요구사항
* 자동차 경주에 참가할 참가자의 이름을 입력 받는다.
  * 이름의 구분은 쉼표(,)를 기준으로 구분한다.
  * 5글자 이하의 이름을 입력해야한다.
  * 경주를 진행하기 위해서는 최소 2대의 자동차가 필요하다.
* 경주의 횟수를 입력한다.
* 사용자가 잘못된 값을 입력할 경우 "[ERROR]"로 시작하는 에레 메시지를 출력 후 입력을 다시 받는다.
* 입력한 횟수 동안 자동차는 경주를 한다.
  * 자동차는 전진 또는 멈춮 수 있으며 조건은 아래와 같다.
    * 0에서 9사이의 random 값이 4 이상일 경우 전진, 3 이하일 경우 멈춰있는다.
* 입력한 횟수 만큼의 경주가 끝난 후 우숭자를 알려준다.
  * 우승자를 출력해 준다. 우승자가 한명 이상일 경우 쉼표(,)로 이름을 구분해 출력 해준다.

## 기능
* 유효성 체크(GameValidator)
  * 사용자 입력한 값을 검증한다.
    * 쉼표로 분리된 자동차의 이름이 5글자 이하인지 검증한다.
    * 레이싱을 진행하기 위해서는 2대 이상의 자동차가 필요하다.
  * 성공여부를 담아서 리턴한다.
  * 사용자가 입력한 회수를 검증한다.
    * 최소 1회이상을 수행한다.
* 자동차(Car)
  * 사용자가 입력한 값과 최초거리 0으로 초기화 한다.
  * 한 바퀴가 진행 될 때마다, 전진(MOVE), 멈춤(STOP)를 넣어준다.
  * 자동차가 현재까지 이동한 위치를 출력해준다.
