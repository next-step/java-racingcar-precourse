# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할기능목록을
### 1. 출력메세지 및 상수 클래스 정의
### 2. 사용자 클래스 정의
> 2-1. 사용자 입력
### 3. 자동차 클래스 정의
> 3-1. 자동자 이름 필드값 getter, setter  
> 3-2. 자동차 이름 유효성 검사(5자리)  
> 3-3. 다수의 자동차 이름 "," 로 나누는 기능  
> 3-4. 멈춤 기능  
> 3-5. 전진 기능  
> 3-6. 멈춤/전진 판단 처리  
### 4. 게임 클래스 정의
> 4-1. 게임 세팅 초기화 (자동차, 사용자)  
> 4-2. 게임 시작 기능  
> 4-3. 경기 횟수 판단 기능  
> 4-4. 우승자 판단 기능  
> 4-5. 우승자 출력 
### 5. 에러처리
