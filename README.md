# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

---

## 기능 요구사항
* 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
* 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
* 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
* 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
* 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
* 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한 명 이상일 수 있다.

---

## 기능목록
1. 자동차 이름들을 입력 받음
    1. 각 이름을 가진 자동차를 생성 한다.
       1. 하나의 자동차 이름은 1자 이상 5자 이하이며, 쉼표(,)는 사용할 수 없다.
       2. 최초 이동거리가 0인 자동차가 생성 된다.
    3. 쉼표(,)를 구분자로 한줄로 입력 받는다.
    4. 하나 이상의 자동차를 입력해야 한다.
    5. 각 자동차의 이름은 유일해야 한다.
2. 경주 횟수를 입력 받음
    1. 1이상의 정수로 한줄로 입력 받는다.
3. (2)에서 입력받은 횟수 만큼 경기를 진행 = (4)-(5)를 반복함
4. 자동차들이 이동함
    1. 이동조건
        1. 각 자동차 마다 랜덤수를 생성 : 4이상 = 전진 or 3이하 = 정지
5. (4)의 이동결과를 출력함
    1. 참가한 모든 자동차에 대한 이동거리를 출력한다.
6. 우승자를 출력함
    1. 우승자는 한 명 이상일 수 있다.
