# 자동차 경주 게임

## 구현할 기능 목록
* 사용자가 입력한 값으로 자동차 생성을 구현  
    * ','기준으로 구분하여 각각 생성  
    * 각각의 이름은 5자 이하  
    * 자신의 이름을 반환할 수 있다.  
* 랜덤값으로 자동차 이동을 구현  
    * 값이 0~3일경우 정지  
    * 값이 4~9일경우 전진  
    * 사용자가 입력한 횟수만큼 움직인다.  
    * 움직인 총 길이값을 반환할 수 있다.  
* 경주 결과 산정  
    * 가장 많이 전진한 자동차를 우승자로 선정한다.  
    * 우승자의 이름을 반환할 수 있다.  
    * 우승자가 여러명일경우 쉼표로 이름 구분해서 반환  
  

## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
