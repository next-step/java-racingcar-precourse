# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.



## 요구사항

- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다. 
- 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력 할때 자동차 이름을 같이 출력한다. 
- 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다. 
  - error: 이름이 5자 초과, 이름이 없음
- 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다. 
  - error: 0이하, number 아님
- 전진하는 조건은 0에서 9사이에서 random값을 구한 후 random값이 4이상일 경우 전진하고, 3이하의 값이면 멈춘다. 
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다. 우승자는 한명 이상일 수 있다.

-------

- 경주 생성기: 
  - 자동차 입력: 문자열, 쉼표 구분, 5자 이상 규칙, 자동차 생성
  - 턴 진행, 자동차 전진 
  - 우승자 판별 및 출력
- 자동차: car
  - 전진: (랜덤) next, 이름과 함께 전진상태 출력
  - 이름 부여



## 프로그래밍 요구사항

- 1주차 요구사항
  - 코드 컨벤션
  - 들여쓰기 depth를 1까지만
  - stream 금지, 람다 가능
  - else 금지
  - 함수 길이 10라인까지 + 한 가지 일만 잘 하도록
- 일급콜렉션 활용
  - 콜렉션을 포함한 클래스는 반드시 다른 멤버 변수가 없어야 한다. 
- 모든 원시값과 문자열 포장
