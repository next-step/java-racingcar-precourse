# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
구현할 기능 정리
1. 자동차 이름 입력받은 후 처리
 - 쉼표를 기준으로 5자리 이하의 이름을 나눈다.
 - 각 car객체에 나눈 이름을 부여한다.
 - 위의 형식을 맞추지 않는 입력값은 예외 처리한다.
 - 예외 처리 : 5자리 초과의 이름, 쉼표외의 특수기호를 사용하거나 형식이 다른경우
2. 사용자는 몇번 이동할지 입력 처리
 - 숫자 아닌 경우 예외 처리
3. 0~9 사이의 랜덤값 구하기
 - 0과 9 사이의 숫자를 구해주는 객체를 생성하여 랜덤값을 구한다.
4. 위에 구한 랜덤값을 기준으로 이동 로직 처리
 - 랜덤값이 4이상이면 전진
 - 랜덤값이 3이하일 경우 멈춤
5. 우승자 정한 후 우승자 출력
 - 전부 다 맞았을 경우, 1을 입력시 게임 지속
 - 단독 우승 :  우승자 출력,
 - 공동 우승 :  쉼표를 기준으로 한 공동 우승자 출력
공통
1. 입력값이 잘못 될 경우
 - [ERROR]로 시작하는 에러 메시지 출력
