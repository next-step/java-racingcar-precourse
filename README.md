# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.


## 구현할 기능 목록
* 모델
    * 자동차 리스트 컬렉션
    * 자동차 정보 
        * 자동차 이름
        * 자동차 이동 거리
            * 이동 / 멈춤 
    
* 게임 진행
    * 초기 
        * 사용자에게 자동차 이름 입력 받음
        * 몇번의 랜덤 함수를 진행할지 입력받음
    * 진행 랜덤 함수를 통하여 전진 멈춤 구현
    * 종료 시 승리자 출력
