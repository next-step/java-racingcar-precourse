# 자동차 경주 게임
### 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

### 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

---

## 구현 기능목록 정리
* **경주에 참가하는 자동차들 이름 입력받기**
  * 쉼표(,)로 자동차 이름 구분
  * 자동차 이름의 자리수가 1자 이상, 5자 이하인지 검증
* **경주동안 자동차가 움직이거나 정지할 횟수 입력받기**
  * 입력값이 1보다 크거나 같은 값인지 검증
* **입력 값에 따라서 자동차 전진 또는 정지**
  * 값이 4이상이면 전진, 3이하면 정지
* **매 횟수마다 실행 결과 출력**
* **경주가 종료되면 우승자 출력**
  * 우승자가 한 명 이상일 경우 쉼표(,)로 구분해서 출력
