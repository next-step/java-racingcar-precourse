# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 프로그램 소개
* 자동차들의 이름들과 시도 횟수를 입력받아서,
시도 횟수만큼 무작위로 전진하게하여 우승한 자동차를 출력하는 프로그램

## 진행단계
* 경주할 자동차들의 이름 입력
* 시도할 횟수 입력
* 자동으로 자동차들의 전진/멈춤을 적용
* 무작위 0~9 값을 자동차 별로 생성하여 4 이상이 경우만 전진함
* 모든시도가 완료된 후 가장 많이 전진한 자동차들이 우승함

## 기능목록
* 입력
  * 경주차들의 이름 입력 (5자 이내의 ','구분 입력)
    * 5자 초과 되거나, 빈 이름 입력시 exception 발생 
  * 시도할 횟수 입력 (1 이상 10 이하 숫자 입력)
    * 숫자 범위를 넘거나, 문자 입력시 exception 발생
* 경주
  * 0~9 사이의 무작위 숫자 생성
  * 4이상 여부에 따른 전진/멈춤 비교
  * 자동차들의 상태 변경
  * 시도 횟수만큼 반복
  * 우승 자동차 판별
* 출력
  * 횟수별 경주상태 출력
  * 우승결과 출력
