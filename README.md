# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 게임 설명
* 사용자에게 경주할 자동차 이름들을 입력받는다.
* 사용자에게 자동차가 완주하기 위해 몇번을 이동해야 하는지 입력받는다.
* 레이싱 게임을 반복하여 가장 먼저 완주에 성공한 자동차가 무엇인지 출력한다.

## 기능 목록

|#|기능 목록|
|---|---|
|1|사용자가 자동차들의 이름을 입력|
|2|사용자가 완주를 위해 몇 번의 이동을 할 것인지 입력|
|3|0~9 사이의 Random 값을 구한 후 값이 4 이상인 경우 자동차 전진|
|4|레이싱 중인 자동차의 이름과 몇번 전진했는지 출력|
|5|레이싱 중인 자동차 중 완주한 자동차 찾기|
|6|완주한 자동차가 있는 경우 자동차의 이름을 화면에 출력|
