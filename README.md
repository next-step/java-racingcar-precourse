# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 구현 목록

* 경주 할 자동차 이름 Input View 단에서 입력(UI)
* 입력 받은 경주 자동차 입력 검증
    1. 최소 2개의 자동차를 입력했는가?
    2. 자동차의 이름이 5자 이하인가?
    3. 이름이 중복되는 자동차가 존재하는가?
    4. 공백이거나 null 값으로 들어오는가?
* 시도 할 회수를 Input View 단에서 입력(UI)
* 입력 받은 회수 검증 
    1. 공백이거나 null 값으로 들어오는가?
    2. 숫자형이 맞는가?
* 난수값에 따라 자동차 전진 여부 판단
    1. 0~4 이상일 경우 true, 3 이하의 값이면 false
* 난수값에 따라 자동차 전진 처리
* 0~9 사이 난수 값 생성 로직 구현
* 현재 전진 실행 결과 Output View 단에서 출력(UI)
* 시도 할 횟수 이 후 결과 로직 구현
    1. 우승한 경주 자동차 판단 로직 구현
    2. 우승 결과 Result View 단에서 출력(UI)
