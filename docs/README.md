[구현 기능 목록]
1. 사용자의 입력을 받는다.
2. random 값 생성
3. 경주 할 자동차 이름, 이동횟수 세팅, 경주 자동차 그룹화
    - 자동차 이름은 쉼표(,)를 기준으로 구분
    - 자동차 이름 5자이하
    - 전진 자동차 출력 시 이름 출력
    - 상태 및 기록을 저장
4. random 값 기준 3이하 정지, 4이상 전진 기능 추가
5. 우승자 출력
   - 우승자 한 명 이상일 경우, 쉼표 구분해 출력
6. 사용자가 정상적이지 않은 입력을 했을 때 IllegalArgumentException을 발생
   - "[ERROR]" 메세지 출력 후 재 입력
