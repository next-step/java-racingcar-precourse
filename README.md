# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 구현 목록
- 경주에 참여할 자동차 이름을 입력받을 수 있어야 한다.
- 자동차 이름은 쉼표로 구분하며, 5자 이하만 가능하다.
- 몇 번을 이동할 것인지 입력받을 수 있어야 한다.
- 각 자동차는 0~9 사이의 random 값을 하나 뽑아 그 값이 4 이상일 경우 전진하고, 3 이하일 경우 움직이지 않는다.
- 사용자가 입력한 값만큼 반복하여 우승자를 출력한다.
