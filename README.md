# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 요구 사항
* 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
* 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차는 출력할 때 자동차 이름을 같이 출력한다.
* 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하만 가능하다.
* 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
* 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
* 자동차 경주 게임을 완료한 후 누가 우승했는지를 알려준다.
* 우승자가 한 명 이상일 경우, 쉼표(,)로 이름을 구분해 출력한다.
* 사용자가 잘못된 값을 입력한 경우 "[ERROR]"로 시작하는 에러 메시지를 출력 후 입력을 다시 받는다.

## 기능 구현 목록
* 자동차
  * 자동차는 이동 거리를 갖는다.
  * 자동차는 이름을 갖는다. (사용자 입력)
  * 자동차의 이동거리는 매 라운드마다 random 값에 따라 전진 혹은 정지 한다.

* 사용자 입력
  * 사용자는 경주할 자동차 이름을 입력할 수 있다.
    * 이름은 쉼표(,)를 기준으로 구분한다.
    * 이름은 5자 이하여야 한다.
  * 사용자는 몇 번의 이동을 할 것 인지 입력할 수 있어야 한다.
    * 이 때 입력하는 문자열은 숫자 유형이어야 한다.
  * 사용자가 잘못된 값을 입력한 경우 "[ERROR]"로 시작하는 에러 메시지를 출력 후 입력을 다시 받는다.

* 콘솔 출력
  * 우승자 출력
  * 에러 메시지 출력
  * 매 라운드(입력 받은 횟수가 증가)마다 자동차의 이름과 현재 거리를 출력한다
