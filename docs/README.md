# 구현할 기능
* 입력
    * 자동차 이름은 5자 이하이다.
    * 자동차 이름 쉼표(,) 기준으로 여러대 입력 받을 수 있다.
    * 사용자는 몇 번의 이동을 할 것인지 입력한다.
        * 숫자만 입력 가능하다.
    * 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시키고, "[ERROR]"로 시작하 에러 메시지를 출 후 그 부분부 입력을 다시 받는다.
* 출력
    * 각 자동차의 이동거리만큼 "-" 를 출력한다.
    * 우승자 2명 이상일 경우 쉼표(,)로 이름을 구분해 출력한다
* 자동차
    * 0~9 사이 random 값을 구하고 4 이상일 경우이면 전진하고, 3이하이면 멈춘다.
* 경주
    * 이동가능 수 만큼 자동차를 전진시킨다.
    * 참여한 자동차 중 이동거리가 가장 큰 자동차들을 찾는다.
