# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 목록
- 랜덤한 숫자에 따라서 1 증가시킬지 말지 결정하는 `RacingBar`를 생성한다.
- 자동차가 3이상일 경우 멈춤, 4이상일 경우 달림을 구현한다.
- 자동차들을 달리게 하는 주체를 만든다. 생성시 자동차에게 이름을 부여하고, 유효성 검증을 한다. 충족하지 않을 경우 Exception이 발생한다.
- 이 주체가 소속된 자동차들을 달리게 한다.
- 자동차들의 경주 진행 상태를 보여주는 기능을 구현한다.
- 랭킹을 반환한다.
- 이름과 시도횟수를 받아서 자동차들을 생성하는 주체를 만든다.
- 횟수만큼 달리게 한다.
- 결과 순위를 반환한다.
