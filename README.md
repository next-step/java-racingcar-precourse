# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)


---
## 구현할 기능 목록
- [입력] 자동차 이름 입력
    - 5자 초과일시 예외처리(숫자, 특수문자, **공백 미허용**)
- [입력] 시도 횟수 입력
    - 0이하의 숫자, 문자 입력 시 예외처리 (1이상의 정수만 입력 가능)


- [시도 결과] 자동차별 랜덤 숫자 생성
- [시도 결과] 자동차별 전진 or 스탑 비교
- [시도 결과] 자동차별 출력


- [경기 종료] 최종 우승자 비교
- [경기 종료] 최종 우승자들 이름 출력
