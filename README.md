# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 제약사항
- Java Code Convention 지키자
- indent 1까지
- steam api 사용 금지
- else 등 나머지 사용 금지
- 메서드 라인 10이하
- ** 메서드는 한가지 일만 하자

## 기능 요구사항
- 자동차는 움직일 수 있다
  - 전진
  - 정지
- 사용자 입력
  - 쉼표(,) 구분으로 자동차 이름을 입력
  - 이동 횟수 입력
- 사용자 입력 예외처리
  - 잘못된 값 일력시 "[ERROR]" 출력 후 에러 메시지 출력
  - 0~9 사이의 숫자인지 확인
- 자동차 이동여부
  - 4이상 9이하일 경우 전진
  - 0이상 3이하일 경우 정지
- 우승자 판별
  - 우승자가 한 명일 경우 우승자 출력
  - 우승자가 여러 명일 경우 쉼표로 구분하여 출력
